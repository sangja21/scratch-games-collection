# 수리검 던지기 (shuriken-throw) — Plan

> 닌자가 화면 하단에서 좌우 이동하며 위로 수리검을 던져, 상단을 좌우로 왕복하는 표적(과녁/두루마리)을 격파한다.
> 베이스: `games/balloon-shooter/build.py` (좌우 스폰 + 클론 + pop 사운드) + `games/zombie-shooter/build.py` (스페이스/클릭 입력 + 발사체 클론). 학습 콘셉트 없음. 초등학생용 직관적 슈팅.

## 1. 컨셉 / 메커닉

- 시점: 정면 (2D). 닌자 = 화면 하단, 표적 = 화면 상단.
- **이동**: 화살표 왼쪽/오른쪽 → 닌자 x 위치 ±4 / 틱. -210 ~ 210 clamp.
- **발사**: 스페이스 키 OR 마우스 클릭 → 수리검 클론 생성, 닌자 머리 위(닌자 y + 20)에서 출발해 위로 직진.
- **수리검 회전**: 클론 매 틱 `motion turn cw 25` (회전 스타일 = all around). 시각적 회전 효과.
- **표적**: 클론으로 등장, 화면 상단 (y = 80 ~ 150). 좌→우 또는 우→좌로 왕복 (가장자리 닿으면 방향 반전). 종류 3가지:
  - 과녁 (크기 70, 점수 10, 속도 빠름)
  - 두루마리 (크기 90, 점수 15, 속도 중간)
  - 황금 종 (크기 60, 점수 30, 속도 빠름, 드물게 등장)
- **라운드 진행**: 표적 N개 처치 → 다음 라운드. 라운드 1 = 5개, 라운드당 +2개, 표적속도 ×1.15.
- **시간제한 없음**, 라운드 무한 진행. 표적은 화면 끝까지 못 가도 OK (왕복하니까 안 사라짐).
- **사격 쿨다운**: 0.22초 (연사 방지).
- 패배 조건 없음 (캐주얼). 점수와 라운드만 누적. 라운드 10 도달 시 "마스터!" 배너.

## 7. 스프라이트별 블록 흐름

### Stage
1. **깃발 클릭** → 모든 변수 초기화 → 0.5초 대기 → `게임시작` 방송.
2. **게임시작 받으면 (스폰 루프)** → forever:
   - if 게임상태 = 1 AND 스폰수 < 라운드목표:
     - 표적종류 = random 1..10 → 1~6=과녁, 7~9=두루마리, 10=황금종 (확률 6:3:1)
     - 표적방향 = random 1..2 의 2배-3 (→ 1 또는 -1)
     - 표적X시작 = (표적방향 = 1) ? -220 : 220 → `if 표적방향=1` 두 갈래
     - 표적Y시작 = random 70..150
     - `표적스폰` 방송 → 스폰수 += 1
   - wait (1.4 - 라운드 * 0.08) 초, 최소 0.5
3. **다음라운드 받으면** → 라운드 += 1 → 표적속도 ×1.15 → 라운드목표 += 2 → 처치수 = 0 → 스폰수 = 0 → 1.5초 대기.
4. **마스터 받으면** → 게임상태 = 2.

### 닌자
1. **깃발 클릭** → 위치 (0, -130), 크기 80, 코스튬 idle, 회전스타일 = `left-right`, 방향 = 90, 보이기.
2. **forever (이동)**:
   - if 왼쪽 화살표 → change x by -4
   - if 오른쪽 화살표 → change x by 4
   - clamp: if x < -210 → set x to -210, if x > 210 → set x to 210
3. **forever (사격 입력)** — 별도 스크립트:
   - if (mouse down OR space pressed) AND 게임상태 = 1 →
     - 코스튬 = throw, 수리검X = x 위치, 수리검Y = y 위치 + 20
     - `사격` 방송 → 0.1초 대기 → 코스튬 = idle → 0.12초 대기 (총 쿨다운 0.22)

### 수리검
1. **깃발 클릭** → 숨기기, 크기 50, 회전스타일 = `all around`.
2. **사격 받으면** → 클론 생성 (master 자신).
3. **클론으로 시작**:
   - 수리검X, 수리검Y 로 이동 → 보이기 → 방향 90 → pop sound (pitch 250)
   - repeat until (y > 180 OR touching 표적):
     - change y by 14
     - turn cw 25 degrees
     - wait 0.02
   - 숨기기 → 클론 삭제

### 표적
1. **깃발 클릭** → 숨기기.
2. **표적스폰 받으면** → 클론 생성.
3. **클론으로 시작**:
   - 내종류 = 표적종류 (변수 복사)
   - 기본: 내점수 = 10, 내속도 = 표적속도, 크기 70, 코스튬 = target
   - if 내종류 = 2 → 코스튬 = scroll, 크기 90, 내점수 = 15, 내속도 = 표적속도 * 0.85
   - if 내종류 = 3 → 코스튬 = bell, 크기 60, 내점수 = 30, 내속도 = 표적속도 * 1.2
   - 내방향 = 표적방향, go to (표적X시작, 표적Y시작), 보이기
   - repeat until touching 수리검:
     - change x by (내속도 * 내방향)
     - if x > 220 → 내방향 = -1
     - if x < -220 → 내방향 = 1
     - if 게임상태 ≠ 1 → 숨기기 + 클론 삭제 (조건 종료)
     - wait 0.025
   - on hit: 코스튬 = hit, 점수 += 내점수, 처치수 += 1, pop sound pitch 0
   - 0.18초 대기 → 숨기기 → 클론 삭제
   - if 처치수 ≥ 라운드목표 →
     - if 라운드 ≥ 10 → `마스터` 방송
     - else → `다음라운드` 방송

### 라운드배너
1. **깃발 클릭** → 숨기기, 위치 (0, 0), 코스튬 round.
2. **다음라운드 받으면** → 코스튬 round → 보이기 → 0.9초 대기 → 숨기기.
3. **마스터 받으면** → 코스튬 master → 보이기 → 영구 유지.
