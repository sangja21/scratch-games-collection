# 로봇 슈터 (robot-shooter) — Plan

> **탑뷰 아레나 로봇 슈터 (목업 UI 1차)**. 내 모빌슈트 1기로 마우스 조준·빔 연사하며, 웨이브마다 몰려오는 적 기체를 전멸시킨다. 웨이브 클리어마다 **로그라이크 강화 택1**, 다음 웨이브는 적 수·체력·속도가 단계적으로 상승.
> 베이스: `games/magic-survivor/build.py`(웨이브·강화 택1·클론·한글 손잡이) + `docs/plan-zombie-shooter.md`(마우스 조준·클릭 사격).
> 학습 콘셉트 없음. 초등 카툰 로봇. 1차는 **플레이 가능한 목업** — HUD는 변수 모니터 + 강화카드/게임오버 SVG 패널.

- **주제**: 탑뷰 아레나 · 웨이브 · 로그라이크 강화
- **카테고리**: 액션
- **난이도**: ★★★☆☆
- **폴더**: `games/robot-shooter/`
- **출력 파일**: `로봇_슈터.sb3`
- **빌드**: `python3 games/robot-shooter/build.py`

## 1. 한 줄 룰

화살표로 로봇 이동, 마우스로 조준, 클릭/스페이스로 빔 연사. 웨이브 목표 수만큼 적을 전멸하면 강화 카드(1~4) 중 하나 선택 → 다음 웨이브는 더 많고 센 적. 체력 0이면 GAME OVER. 점수 = 처치 수.

---

## 2. 화면 / 좌표

- 무대 480×360. 좌표 -240..240 / -180..180.
- 배경: 어두운 격납고/아레나 바닥(그리드) + 코너 장식 (목업).
- HUD(모니터): 체력 · 웨이브 · 점수 · 적수 (좌상단).

```
 (-240,180)                                      (240,180)
   ┌────────────────────────────────────────────┐
   │ 체력  웨이브  점수  적수                     │
   │          [적MS]     [적MS]                  │
   │     [내 로봇 ▶]  ←마우스 조준 빔→           │
   │          [적MS]                             │
   └────────────────────────────────────────────┘
 (-240,-180)                                    (240,-180)
```

## 5. 웨이브 / 스케일

- `웨이브목표 = 웨이브기본수 + (웨이브-1) * 웨이브증가`
- 스폰: 화면 가장자리 랜덤, `스폰간격`마다 1기, `스폰수`가 목표에 도달할 때까지.
- 전멸 조건: `스폰수 >= 웨이브목표` AND `적수 == 0` → 게임상태=2, 강화카드 show.
- 적 스탯 (생성 시):
  - `내체력 = 티어체력 + (웨이브-1) * 적체력증가`
  - `내속도 = 티어속도 + (웨이브-1) * 적속도증가`
- 웨이브 구간 티어 믹스:
  - 1–2: 약만
  - 3–4: 약·중
  - 5+: 약·중·강

---

## 6. 강화 (로그라이크 택1)

| 키 | 효과 |
|----|------|
| 1 | 공격력 + 강화량 |
| 2 | 발사간격 − 0.05 (하한 0.12) |
| 3 | 이동속도 + 강화량 |
| 4 | 추가발사 + 1 (다연장 부채꼴) |

선택 후: 웨이브 +1, 스폰수/적수 리셋, 게임상태=1, 다음 웨이브 시작.
