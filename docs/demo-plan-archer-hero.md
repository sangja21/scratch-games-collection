# 데모: 아처 용사 (archer-hero)

> **데모 스코프** — 풀게임 아님. 핵심 루프의 "손맛"만 검증받기 위한 빌드. 폴리시 생략. 목표 규모 300~500 블록.

## 콘셉트
Archero(궁수의 전설) 형 **탑뷰 "멈추면 발사" 슈터**. 화살표로 움직이는 동안엔 발사 불가, **멈추면 가장 가까운 적에게 자동 연사**. "지금 피할까, 서서 쏠까"의 리듬이 핵심 재미. 조준 불필요라 조작은 이동뿐 — 초등학생에게 가장 쉬운 조작 + 긴장감.

## 화면/스프라이트 (7개 + Stage)
- **Stage**: 옅은 던전 아레나 배경. 웨이브 스폰 관장.
- **용사(궁수)**: 화살표 4키 8방향 이동 (이동속도 5.5). **아무 화살표도 안 눌린 상태면 0.25초마다 최근접 적에게 화살 발사**. 이동 중엔 발사 정지. 발사 가능 상태를 시각화 (조준선 or 캐릭터 자세 코스튬 전환 — 멈추면 활 겨눔).
- **화살**: 클론. 발사 시 최근접 적 방향으로 직진 (화살속도 11). 명중/화면 밖 삭제.
- **적**: 클론 2종 코스튬 — 걷는 놈(플레이어 향해 직진, 속도 2.2), 돌진 놈(1.5초마다 플레이어 방향 대시). 잡졸체력 2. 접촉 피해 1 + 무적 0.5초.
- **강화카드**: 웨이브 클리어 시 3장 (공격력 / 연사속도 / 이동속도). 클릭 폴링+디바운스.
- **체력바**: 11단 costume-fill.
- **웨이브표시**: 변수 모니터 (웨이브 N).

## 웨이브 구조 (데모: 3웨이브면 충분)
- 웨이브 N: 적 4+2N 마리 스폰 → 전멸 시 강화 택1 → 다음 웨이브 (적 체력·속도 × 적성장배율^N)

## 핵심 변수 (한글 손잡이 ~12개)
이동속도 5.5 / 발사쿨 0.25 / 화살속도 11 / 공격력 1 / 잡졸체력 2 / 적속도 2.2 / 돌진속도 6 / 내체력 5 / 무적시간 0.5 / 웨이브기본수 4 / 적성장배율 1.25 / 정지판정 (멈춤 인식 틱)

## 템포
- 멈추면 즉시(0.25초 내) 첫 발 발사 — "멈춤=발사" 인과가 몸으로 느껴져야 함
- 잡졸 2발, 웨이브 하나 20~30초

## 사운드/연출 (데모 최소)
- 합성 효과음 3종: 활시위(twang), 처치(pop), 피격(hurt)
- 멈춤 상태 시각 피드백 필수 (겨눔 코스튬/조준선) — "왜 안 쏘지?" 혼란 방지 (chess-war UI 교훈)

## 검증 (행동 시나리오 — headless)
1. **키 안 눌림 → 발사 시작 / 키 눌림 → 발사 정지** (핵심 행동, 반드시 실측)
2. 화살이 최근접 적 방향으로 실제 이동, 명중 시 HP 감소, 2발 처치
3. 적 접촉 → 피해 + 무적
4. 웨이브 전멸 → 강화카드 → 선택 시 스탯 변화 → 다음 웨이브 적 강화(지수)
5. 자동 플레이(이동-멈춤 반복) 웨이브 1 클리어
