## 빌드 절차 (loop 한 사이클이 수행)

1. status=pending 첫 게임 선택.
2. status=in_progress 로 마킹.
3. `scratch-planner` → `docs/plan-{slug}.md`
4. `scratch-builder` → `games/{slug}/build.py` + `.sb3`
5. `scratch-verifier` → 검증
6. PASS/WARN → README.md 표에 추가 + status=done + git commit + push origin main
7. FAIL → status=failed:{사유}, 커밋 금지
