# 💰 Compound Cards — 웹게임 기획서

> **장르**: 푸시-유어-럭(Push-your-luck) 카드 게임
> **플랫폼**: 웹 (브라우저, 모바일 친화)
> **세션 길이**: 1판 2~3분 / 데일리 챌린지 모드
> **포지션**: *교육용 아님* — 가벼운 갬블 카드 게임. 원래 Scratch에서 "복리 학습 게임"으로 출발했으나, 메커닉이 그 자체로 재미있어 일반 웹게임으로 발전시킨다.
>
> **상태**: 디자인 문서. 미구현. (Scratch 원본은 삭제됨, git 히스토리에 남아 있음)
> **작성일**: 2026-05-18

## 한 줄

자산 1000원으로 시작. 10턴 동안 매 턴 카드 3장 중 1장을 골라 자산을 굴려, 목표 자산(10000원)에 도달하면 승리. 카드는 "안전한 +5%" 부터 "도박급 +100%, p=25%" 까지 다양.

핵심은 **위험-수익 트레이드오프**와 **운의 누적**. 안전 카드만 골라서는 절대 목표 도달 불가능 → 어느 시점에 위험 카드를 끼워 넣을지가 전부.

## 왜 재미있는가 (설계 의도)

- **즉각적 결정**: 매 턴 3장 중 1장. 3초 안에 손이 가는 결정.
- **롤러코스터 곡선**: 50% 도박 카드 한 방에 자산이 두 배 또는 반토막. 매 라운드 *심장이 뛴다*.
- **수학적 정직함**: 모든 카드의 기댓값(E[r])이 라벨에 노출됨. "기댓값 양수만 골라도 안전한가?"라는 질문이 자연스럽게 떠오르고, 답은 "아니오 (분산 때문)".
- **짧은 세션 × 높은 재플레이성**: 1판 2분 + 시드 기반 데일리 챌린지로 친구와 점수 비교.

레퍼런스: *Reigns* (한 결정 = 한 카드), *Slay the Spire* (이벤트 카드의 위험·보상), *Balatro* (수학적 가시화), *Press Your Luck* (게임쇼).

## 코어 루프

```
[게임 시작] → 자산 1000, 턴 1/10
   ↓
[카드 3장 등장] (풀에서 무작위 추첨)
   ↓
[플레이어가 1장 클릭]
   ↓
[주사위 굴림: 확률 p 로 성공]
   - 성공: 자산 × (1 + r)
   - 실패: 자산 × 0.5
   ↓
   ┌─ 자산 ≥ 10000? → 승리
   ├─ 턴 = 10?      → 패배
   └─ 그 외          → 다음 턴
```

## 카드 풀

| # | 이름             | r (성공 시) | p (성공률) | E[r]    | 톤        |
|---|------------------|-------------|------------|---------|-----------|
| 1 | 채권             | +5%         | 100%       | +5.0%   | 안전      |
| 2 | 우량 회사채      | +8%         | 95%        | +5.1%   | 안전      |
| 3 | 우량주           | +12%        | 85%        | +2.7%   | 무난      |
| 4 | 성장주           | +20%        | 70%        | -1.0%   | 위험      |
| 5 | 벤처             | +35%        | 55%        | -3.5%   | 위험      |
| 6 | 투기             | +50%        | 40%        | -10.0%  | 도박      |
| 7 | 코인             | +100%       | 25%        | -12.5%  | 도박      |
| 8 | 현금 보관        | -3%         | 100%       | -3.0%   | 패널티    |

> **밸런스 노트**: 안전 카드만 매 턴 골라도 `1000 × 1.05^10 ≈ 1629` — 목표(10000) 도달 불가능. 도박 카드 최소 1~2장이 끼어야 함.

> **확장**: 무드 카드(보너스/디버프, 한 턴 제한), 이벤트 카드(전체 카드 r ±5%, 한 턴), 부적/유물(영구 효과) 로 메타 진행 가능.

## HUD

- 자산 (큰 숫자, 변화 시 카운트업 애니메이션)
- 턴 (3/10)
- 목표 자산 (10,000)
- **필요 평균 수익률**: `(목표/현재)^(1/남은턴) - 1` — 매 턴 갱신. "지금 페이스로는 안 됨" 을 직관적으로 보여줌.
- 마지막 결과 토스트: "+250 ✓" 또는 "−500 ✗" 짧은 애니메이션

## 게임 모드

### 1. 표준 (Standard)
- 풀 무작위. 카드 3장 매 턴 새로 뽑힘.
- 1판 끝나면 점수(최종 자산) 기록.

### 2. 데일리 (Daily Challenge)
- 시드 기반. 전 세계 모두가 같은 카드 시퀀스로 플레이.
- 결과를 친구에게 공유 ("Day 187: 12,450 / 7턴 클리어").
- 리더보드 (선택적).

### 3. 모디파이어 (Modifier)
- 시작 자산 변경 (500 / 1000 / 2000)
- 목표 변경 (5x / 10x / 20x)
- 턴 수 변경 (5 / 10 / 15)
- 실패 패널티 변경 (×0.5 / ×0.7 / ×0)
- 안전 카드 제외 모드 ("도박 전용")

### 4. 엔드리스 (Endless)
- 턴 제한 없음. 매 5턴마다 목표 자산 2배.
- 살아남는 한 계속. 결국 ×0.5 한 방에 죽음.

## 사운드 / 피드백

- 카드 클릭: 카드 뒤집는 *flip* 소리
- 성공: 동전 짤랑 + 자산 카운트업
- 실패: 낮은 *thud* + 자산 카운트다운
- 승리: 팡파레 + 컨페티
- 패배: 잔잔한 피아노 + 최종 자산 표시
- 옵션: 무음 모드 (도서관 친화)

## 데이터 모델 (TypeScript 스케치)

```typescript
type Card = {
  id: string;        // "bond_safe", "venture", etc.
  name: string;      // 표시명 (i18n)
  r: number;         // 성공 시 수익률 (예: 0.15)
  p: number;         // 성공 확률 (0~1)
  failMult?: number; // 실패 시 곱수 (기본 0.5)
  rarity?: "common" | "rare" | "legendary"; // 확장용
};

type GameState = {
  mode: "standard" | "daily" | "modifier" | "endless";
  seed?: string;
  cash: number;
  goal: number;
  turn: number;
  maxTurn: number;
  history: TurnResult[];
  status: "playing" | "won" | "lost";
};

type TurnResult = {
  turn: number;
  cardId: string;
  success: boolean;
  cashBefore: number;
  cashAfter: number;
};
```

## 확장 아이디어 (백로그)

- **메타 진행**: 판 클리어 시 통화(보석 등) 획득 → 신규 카드/모디파이어/스킨 언락. Roguelite 요소.
- **부적/유물 시스템**: "다음 실패 1회 무효", "도박 카드 +10%p 확률" 같은 영구 효과. *Slay the Spire* 모티프.
- **PvP 모드**: 같은 시드를 두 명이 동시 플레이, 점수 비교. WebSocket 또는 단순 동기 비동기 비교.
- **카드 합성**: 사용한 카드를 합쳐 새 카드 제작.
- **스토리 모드**: 직장인 → 부자 되기, NPC 카드 추천 등 캐릭터성 추가.
- **다크/라이트/세피아 테마**.

## MVP 범위 (최초 출시)

1. 표준 모드 1개.
2. 카드 풀 8종 고정.
3. 자산/턴/필요수익률 HUD.
4. 데일리 챌린지 (시드 = `YYYY-MM-DD` 해시).
5. localStorage 베스트 점수.
6. 컨페티 + 사운드.

이 정도면 1~2주 만에 완성 가능한 사이즈. 메타 진행/유물/PvP 는 모두 v2 이후.

## 주의사항

- **갬블링 우려**: 실제 돈 없음, 광고 없음, "다시 도전" 자유. 미성년자 노출 시 부모 안내 문구 권장.
- **밸런스 검증**: 카드 풀 변경 시 시뮬레이션 10만 판 돌려 승률 5~20% 사이인지 확인 (너무 쉬우면 노잼, 너무 어려우면 좌절).
- **다국어**: 영어/한국어 우선. 카드 이름은 짧고 직관적으로.
