# 좀비 슈터 (zombie-shooter) — Plan

> 톱다운 트윈스틱 슈터. 화살표키로 이동 + 마우스로 조준/사격, 다가오는 좀비 클론을 처치하며 웨이브 생존.
> 베이스: `games/alien-invasion/build.py` (클론·발사·웨이브) + `games/duck-hunt/build.py` (마우스 조준 + 클릭 사격 + 섬광/사운드).
> 학습 콘셉트 없음. 좀비는 만화풍·귀여움 (눈 두 개, 미소, 녹색). 잔인성·공포감 X.

## 1. 컨셉 / 메커닉

- 톱다운 시점, 480×360 무대 위에서 플레이어가 자유 이동.
- **이동**: 화살표키 위/아래/왼쪽/오른쪽 (8방향 — 두 키 동시 누르면 대각선). WASD 도 지원.
- **조준**: 플레이어는 항상 마우스 포인터 방향을 향함 (`point towards mouse-pointer`).
- **사격**: 마우스 클릭(또는 스페이스) → 총알 클론 생성 → 플레이어 위치에서 마우스 방향으로 발사.
- **좀비 AI**: 클론으로 등장, 화면 가장자리에서 스폰, 매 틱 플레이어 방향으로 회전 후 `좀비속도` 만큼 전진. 단순한 follow AI.
- **충돌**:
  - 총알 ↔ 좀비 → 좀비 사망 (점수 +10, 좀비 클론 삭제, 총알 클론 삭제)
  - 좀비 ↔ 플레이어 → 라이프 -1, 좀비 사라짐, 1초 무적
- **웨이브**: 웨이브당 좀비 N마리 처치 시 다음 웨이브 (마리 +3, 속도 ×1.12). 웨이브 1 = 5마리.
- **라이프 3**, 0 되면 게임오버 배너.

## 6. 스프라이트별 블록 흐름

### Stage
1. **깃발 클릭** → 모든 변수 초기화 → 0.5초 대기 → `게임시작` 방송.
2. **게임시작 받으면** → forever:
   - if 게임상태=1 AND 스폰수 < 웨이브목표:
     - 스폰X = random −230..230
     - 스폰Y = random −170..170 (단, 플레이어 근처 100px 이내는 회피 — 단순화: 그냥 화면 가장자리 4면 중 하나 선택)
     - 단순화 채택: random 1..4 (1=상, 2=하, 3=좌, 4=우) → 가장자리에서 스폰
       - 1: SX=random(-230..230), SY=180
       - 2: SX=random(-230..230), SY=-180
       - 3: SX=-230, SY=random(-170..170)
       - 4: SX=230, SY=random(-170..170)
     - `좀비스폰` 방송
     - 스폰수 +1
   - wait `2 / 웨이브` 초 (웨이브가 올라갈수록 스폰 간격 감소, 최소 0.5초)
3. **다음웨이브 받으면** → 웨이브 +1, 좀비속도 ×1.12, 웨이브목표 +3, 처치수=0, 스폰수=0 → 1.5초 대기.

### 플레이어
1. **깃발 클릭** → 보이기, 크기 70, 위치 (0,0), 회전 스타일 = `all around`, 코스튬=생존, 무적시간=0.
2. **forever (이동 + 조준)**:
   - 마우스 방향으로 point (`motion_pointtowards` mouse-pointer)
   - if 위(또는 W) 키 → change y by 3
   - if 아래(또는 S) 키 → change y by -3
   - if 왼쪽(또는 A) 키 → change x by -3
   - if 오른쪽(또는 D) 키 → change x by 3
   - 가장자리 clamp: if x>230 → set x to 230, if x<-230 → -230, y 동일하게 ±170
   - if 무적시간 > 0 → 무적시간 -1 (틱 단위)
3. **forever (사격 입력)** — 별도 스크립트:
   - if 마우스 down OR 스페이스 키 → `사격` 방송 → 0.18초 대기 (rate limit)
4. **사격 받으면** → 총알X = x position, 총알Y = y position, 총알방향 = direction (방금 마우스 방향), pop 사운드 재생, 총알 클론 생성.

### 총알
1. **깃발 클릭** → 숨김, 크기 50.
2. **클론으로 시작**:
   - 총알X/Y 로 이동 → 총알방향으로 point → 보이기
   - repeat until (x<-240 OR x>240 OR y<-180 OR y>180 OR `touching 좀비`):
     - move 10 steps
   - 숨김 → 클론 삭제

### 좀비
1. **깃발 클릭** → 숨김, 크기 70, 코스튬=zombie1, 회전 스타일 = `all around`.
2. **좀비스폰 받으면** → 클론 생성.
3. **클론으로 시작**:
   - 스폰X/Y 로 이동, 보이기, 코스튬=zombie1
   - repeat until (touching 총알 OR (touching 플레이어 AND 무적시간=0)):
     - point towards 플레이어
     - move 좀비속도 steps
     - 매 8틱 next costume (걷기 애니메이션) — 단순화: 매 iteration 끝에 50% 확률 next costume
     - wait 0.04
   - if touching 총알: 처치수 +1, 점수 +10, 코스튬=dead, 0.15초 대기, 숨김, 클론 삭제
     - if 처치수 ≥ 웨이브목표 → `다음웨이브` 방송
   - else (touching 플레이어): 라이프 -1, 무적시간=20 (약 0.8초), 숨김, 클론 삭제
     - if 라이프 ≤ 0 → 게임상태=0

### 게임오버
- duck-hunt 와 동일 패턴: 깃발→숨김+중앙 위치→게임상태=1까지 대기→게임상태=0까지 대기→show.

## 7. 사운드

- **사격**: pop @ pitch 150 (밝은 "팡")
- **명중**: pop @ pitch 0 (기본 "팡")
- **피격(좀비 닿음)**: pop @ pitch -250 (둔탁한 "퉁")

assets/pop.wav 는 duck-hunt/missile-command 에서 복사.
