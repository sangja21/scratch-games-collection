# 데모: 탕탕 헌터 (tangtang-hunter)

> **데모 스코프** — 풀게임 아님. 핵심 루프의 "손맛"만 검증받기 위한 빌드. 미션 사다리/README/강화 다단계 등 폴리시 생략. 목표 규모 300~500 블록.

## 콘셉트
탕탕특공대/20 Minutes Till Dawn 형 **탑뷰 수동 조준 슈터**. 화살표로 이동하면서 **마우스 방향으로 총알을 연사**한다. 사방에서 몰려오는 적을 쏘아 경험치를 모으고 레벨업 강화를 고른다. magic-survivor(자동공격·회피 전용)와 정확히 반대 — "내가 직접 쏘는 맛"이 셀링 포인트.

## 화면/스프라이트 (7개 + Stage)
- **Stage**: 옅은 단색 아레나 배경 (파스텔 그리드). 게임 루프/스폰 관장.
- **플레이어**: 총 든 캐릭터. 화살표 4키 8방향 이동 (이동속도 5). 마우스 방향 바라봄.
- **총알**: 클론. 마우스 버튼 누르는 동안 발사쿨(0.15초)마다 플레이어 위치에서 마우스 방향으로 발사, 총알속도 12, 화면 밖/명중 시 삭제. **눈에 확실히 보이는 크기/색**.
- **적**: 클론. 화면 가장자리 랜덤 스폰, 플레이어 향해 직진 (적속도 2.5). 잡졸체력 2 (2발 처치). 접촉 시 플레이어 피해 1 + 무적 0.5초. 처치 시 팝 이펙트 + 경험치.
- **강화카드**: 레벨업 시 3장 (공격력+1 / 연사속도↑ / 이동속도↑). 클릭 폴링+디바운스.
- **체력바**: 11단 costume-fill.
- **경험치바**: 11단 costume-fill. 만충 시 레벨업.

## 핵심 변수 (한글 손잡이 ~12개)
이동속도 5 / 발사쿨 0.15 / 총알속도 12 / 공격력 1 / 잡졸체력 2 / 적속도 2.5 / 스폰간격 0.8 / 내체력 5 / 무적시간 0.5 / 레벨업경험치 8 / 적성장배율 1.2 (레벨 비례 적 강화 — magic-survivor 교훈) / 경험치획득 1

## 템포 (chess-war 교훈: 처음부터 빠르게)
- 시작 3초 안에 첫 적 등장, 첫 10초 안에 첫 레벨업 가능할 밀도
- 총알 연사감 (초당 ~6발), 잡졸 2발 처치, 화면에 항상 적 5~10마리

## 사운드/연출 (데모 최소)
- 합성 효과음 3종: 발사(pew), 처치(pop), 피격(hurt)
- 처치 시 적 팝 애니메이션(축소+사라짐), 데미지 팝업은 생략 가능

## 검증 (행동 시나리오 — headless)
1. 마우스다운 → 총알 클론 스폰 (발사쿨 간격), 마우스 방향으로 실제 좌표 이동
2. 총알-적 명중 → 적 HP 감소, 2발째 처치 (경험치 증가)
3. 적 접촉 → 0.5초 내 내체력 감소 + 무적 성립
4. 경험치 만충 → 강화카드 표시, 선택 시 해당 스탯 실제 변화
5. 자동 플레이(이동+발사) 30초 생존 가능
