# 데모: 진화 탱크 (evo-tank)

> **데모 스코프** — 풀게임 아님. 핵심 루프의 "손맛"만 검증받기 위한 빌드. 폴리시 생략. 목표 규모 300~500 블록.

## 콘셉트
diep.io 형 **탑뷰 탱크 진화 슈터**. 화살표로 탱크를 몰고, 포탑이 마우스를 따라 돌며, 클릭(홀드)으로 포탄 발사. 필드의 도형(경험치 오브젝트)과 추격해 오는 적 탱크를 부수며 레벨업 → **탱크가 진화**(쌍포신/스나이퍼/머신건 택1, **포탑 모양이 실제로 바뀜**). 진화의 시각 변화가 셀링 포인트.

## 화면/스프라이트 (8개 + Stage)
- **Stage**: 옅은 그리드 배경 (diep.io 감성). 스폰 관장.
- **탱크(본체)**: 화살표 4키 이동 (이동속도 4.5, 살짝 미끄러지는 관성 허용—없어도 됨). 본체는 방향 고정.
- **포탑**: 탱크 위에 붙어 마우스 방향으로 회전. **진화 시 코스튬 교체** (기본 단포신 / 쌍포신 / 스나이퍼 장포신 / 머신건 넓은 포신).
- **포탄**: 클론. 마우스 홀드 시 발사쿨마다 포탑 방향 발사. 진화별 발사 패턴: 기본(1발/0.3초) / 쌍포신(2발 평행) / 스나이퍼(1발/0.6초, 데미지 3배·탄속 1.5배) / 머신건(1발/0.12초, 약간 산탄).
- **도형**: 클론. 사각(HP2·경험치1)/삼각(HP4·경험치3) 필드에 떠 있음. 부수면 경험치.
- **적탱크**: 클론. 가장자리 스폰, 플레이어 추격 + 접촉 피해 1 + 무적 0.5초. 체력 3.
- **진화카드**: 레벨업 시 3장 (쌍포신/스나이퍼/머신건 — 한 번 고르면 데모에선 고정). 클릭 폴링+디바운스.
- **체력바/경험치바**: 11단 costume-fill 각 1개.

## 핵심 변수 (한글 손잡이 ~13개)
이동속도 4.5 / 발사쿨 0.3 / 포탄속도 10 / 공격력 1 / 사각체력 2 / 삼각체력 4 / 적탱크체력 3 / 적탱크속도 2 / 내체력 5 / 무적시간 0.5 / 레벨업경험치 10 / 스폰간격 1.2 / 진화 (0=기본 1=쌍포신 2=스나이퍼 3=머신건)

## 템포
- 시작 즉시 도형 6~8개 배치 — 쏠 대상이 항상 있음
- 첫 레벨업(진화)까지 15~20초 — 진화가 데모의 하이라이트이므로 빨리 보여줌
- 진화 후 발사 패턴 차이가 확 느껴져야 함 (쌍포신 = 화면에 총알 2줄)

## 사운드/연출 (데모 최소)
- 합성 효과음 3종: 발사(pew 낮은음), 파괴(crack), 피격(hurt)
- 도형 파괴 시 조각 팝 이펙트, 진화 순간 플래시/사운드 강조

## 검증 (행동 시나리오 — headless)
1. 마우스 홀드 → 포탄 클론 스폰(쿨 간격), 포탑 방향으로 실제 좌표 이동
2. 포탄-도형 명중 → HP 감소·파괴 → 경험치 증가
3. 적탱크 접촉 → 피해 + 무적
4. 레벨업 → 진화카드 → **선택 후 발사 패턴 실제 변화** (쌍포신: 틱당 포탄 2개 / 스나이퍼: 쿨 0.6·데미지 3 / 머신건: 쿨 0.12) — 핵심 행동, 반드시 실측
5. 자동 플레이 30초 생존 + 진화 1회 도달

## 기술 규칙
클론끼리 상호작용 금지 / vrep 리포터 / 클릭 폴링+디바운스 / say 금지 / add_comment 3~5개
