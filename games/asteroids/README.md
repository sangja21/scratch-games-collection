# 소행성 (Asteroids)

> 우주를 떠다니는 소행성을 우주선으로 격파하는 클래식 액션. 관성으로 미끄러지듯 움직이는 우주선과 화면 가장자리에서 반대편으로 이어지는 wrap이 특징입니다.

## 조작

- 왼쪽/오른쪽 화살표: 우주선 좌/우 회전
- 위 화살표: 분사 (가속도 누적, 손을 떼도 관성으로 계속 미끄러짐)
- 스페이스: 총알 발사 (쿨다운 약 0.4초)
- 화면 가장자리는 반대편으로 이어집니다 (우주선·총알·소행성 모두 wrap)

## 게임 흐름

1. 초록 깃발 클릭 → 우주선이 화면 중앙에 등장.
2. **라운드 1**: 큰 소행성 3개가 화면 가장자리에서 등장. 라운드가 올라갈수록 개수와 속도 증가 (최대 6개).
3. 소행성을 총알로 격파:
   - **큰 소행성** (+20점) → 중간 소행성 2개로 분열.
   - **중간 소행성** (+50점) → 작은 소행성 2개로 분열.
   - **작은 소행성** (+100점) → 소멸.
4. 화면에 남은 소행성이 모두 없어지면 1.5초 후 다음 라운드.
5. 우주선이 소행성에 닿으면 게임오버 배너 표시.
