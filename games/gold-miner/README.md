# 골드 마이너 (Gold Miner)

화면 위쪽 중앙의 광부에 매달린 후크가 좌우로 진자처럼 흔들린다. 흔들리는 타이밍을 보고 **스페이스(또는 클릭)** 를 누르면 후크가 그 순간 가리키던 방향으로 쭉 내려간다. 아이템에 닿으면 잡아서 끌어올리고, 빗나가면 화면 끝까지 갔다가 빈손으로 돌아온다. 무거운 금괴는 점수가 크지만 천천히 올라오고, 가벼운 다이아는 빠르게 올라온다. 돌은 무거운데 점수가 작으니 피하는 게 이득. 60초 안에 라운드 목표 점수를 채우면 다음 라운드로! 1989년 클래식 "골드 마이너" 메커닉을 충실히 재현했다.

## 조작

| 입력 | 동작 |
|------|------|
| 스페이스 / 마우스 클릭 | 후크 발사 (흔들리는 중에만) |
| 초록 깃발 ▶ | 새 게임 시작 / 재시작 |

후크는 광부(0, 120)를 피벗으로 정아래 기준 좌우 ±75°를 사인 곡선으로 부드럽게 왕복합니다. 발사하면 그 순간의 각도가 고정되어 직선으로 하강하므로, **후크가 노리는 아이템을 향할 때 눌러야** 합니다. 한 번 발사하면 잡거나 빗나가서 돌아올 때까지 다시 쏠 수 없습니다.

## 게임 흐름

1. 깃발을 누르면 점수 0 · 라운드 1 · 목표 150점으로 시작하고, 땅속 플레이 영역에 아이템 8~10개가 무작위로 배치됩니다 (라운드가 오를수록 살짝 많아지며 최대 10개).
2. 흔들리는 후크를 보고 타이밍을 맞춰 스페이스를 눌러 발사합니다.
3. 후크가 아이템에 닿으면 그 아이템을 잡아 매단 채 광부 쪽으로 되감기합니다. 무거울수록 천천히 올라옵니다.
4. 후크가 피벗에 도착하면 잡은 아이템의 점수가 더해지고, 후크는 다시 흔들리기 시작합니다.
5. 아무것도 못 잡고 화면 끝(아래·좌·우 가장자리)에 닿으면 빈손으로 빠르게 되감기합니다.
6. 60초가 지나면 라운드를 판정합니다 — 점수가 목표 이상이면 **ROUND CLEAR!**, 라운드가 1 오르고 목표가 +100 되며 아이템이 재배치됩니다. 목표를 못 채우면 **GAME OVER**.
7. 깃발을 다시 누르면 즉시 새 게임이 시작됩니다.

## 룰

- 아이템 5종 — 점수 / 되감기 속도(무거울수록 느림):
  - 금괴 대 200점 (느림) · 금괴 중 100점 · 금괴 소 50점 (빠름) · 돌 20점 (느림·저점) · 다이아 150점 (가장 빠름)
- 한 번 발사에 잡을 수 있는 아이템은 하나뿐입니다 (먼저 닿은 아이템이 잡히면 나머지는 그 발사 동안 잡히지 않습니다).
- 점수는 라운드가 넘어가도 누적되며, 목표 점수도 누적 기준으로 매 라운드 +100 올라갑니다.
- 무거운 대형 금괴는 점수가 크지만 끌어올리는 데 시간이 걸려 그만큼 시간 압박이 큽니다 — "지금 큰 거? 아니면 빠른 작은 거?"를 판단하는 게 핵심 전략입니다.
- 상단 모니터에 점수 · 목표점수 · 시간 · 라운드가 표시됩니다.

## 구조

- 스프라이트 6: Stage(라운드 진행·60초 타이머·아이템 배치·판정) / 광부(피벗 장식 + 되감기 타겟) / 줄(피벗→후크 stretch 표현) / 후크(흔들림·하강·되감기 3상태 머신) / 아이템(5타입 클론) / 배너(클리어·게임오버)
- 총 블록 309개 (Stage 83 / 광부 6 / 줄 14 / 후크 92 / 아이템 93 / 배너 21)
- 핵심 메커닉:
  - **진자 흔들림**: `후크각도 = 180 + 75 × sin(phase)` (phase 매 틱 +4), 피벗(0,120)에서 줄길이 60px 거리에 후크 위치 `(sin·60, cos·60)` 배치
  - **각도 고정 하강**: 발사 순간의 후크각도를 그대로 유지하며 `change x/y by sin/cos(각도)×8`로 직선 이동, 경계(y<-175 또는 x 좌우 끝) 도달 시 빈손 되감기
  - **무게별 되감기**: `point towards 광부 + move 잡힌속도 steps` (금괴대 3 ~ 다이아 11, 빈손 12)로 무게에 따라 속도 차등
  - **잡힘 단일화**: 전역 `잡힘플래그`로 한 발사당 한 아이템만 잡히게 보장, 발사할 때마다 0으로 리셋
  - **가중 랜덤 배치**: `random(1..100)`을 중첩 if-else로 5타입에 분배 (금괴 중심, 다이아·돌 소량)
- 자산: 인라인 SVG 11종 (배경·광부·후크·줄·금괴 대/중/소·돌·다이아·클리어·게임오버) + pop.wav
