# 🚀 다항함수 발사 게임

다항함수 `y = a·x² + b·x` 의 계수 **a**, **b** 를 실시간으로 조절하면서, 그 곡선을 따라 로켓을 발사해 풍선을 터뜨리는 게임입니다.

> **학습 포인트** — 계수가 바뀌면 곡선 모양과 발사체 경로가 어떻게 변하는지를 *수식 / 곡선 / 로켓 비행* 세 가지가 동시에 움직이며 보여줍니다.

## 🎮 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 슈팅 / 학습 게임 |
| 핵심 메커닉 | 곡선 미리보기를 보면서 a, b 맞춰 발사 |
| 학습 주제 | 다항함수 (특히 2차 함수) 의 계수와 그래프 모양 관계 |
| 권장 학년 | 중학교 1~2학년 (1차 함수 / 2차 함수 도입 직후) |
| 플레이 시간 | 라운드당 30초 ~ 1분, 총 무제한 |

## 🕹 조작법

| 키 | 동작 | 변화량 |
|---|------|--------|
| `↑` | 곡선이 더 아래로 휨 (`a` 증가) | +0.002 |
| `↓` | 곡선이 더 위로 휨 (`a` 감소) | −0.002 |
| `→` | 출발 기울기 가파르게 (`b` 증가) | +0.10 |
| `←` | 출발 기울기 완만하게 (`b` 감소) | −0.10 |
| `Space` | 로켓 발사 | — |

> 좌상단 a, b 슬라이더는 마우스로 직접 끌어서 더 빨리 조절할 수 있습니다.

## 📺 화면 구성

```
┌──────────────────────────────────────────────────────┐
│ 점수 0          y = -0.005·x² + 1·x          ☀ │
│ 라운드 1                                              │
│ a ▭━━━━━                                              │
│ b ▭━━━━━           🎈    🎈                          │
│                                                       │
│         ╭┄┄┄┄┄┄┄┄┄┄╮     🎈                          │
│        ┄              ┄                                │
│       ┄                ┄                              │
│      ┄    🎈            ┄                             │
│ 🚀  ┄                    ┄                            │
│════════════════════════════════════════════════════════│  ← 잔디 / 땅
└──────────────────────────────────────────────────────┘
       ↑                              ↑
   분홍 점선 = 발사 곡선 미리보기    풍선 = 표적
```

- **상단 중앙**: 현재 수식이 큰 글씨로 표시 (`y = -0.005·x² + 1·x` 같은 식)
- **좌상단**: 점수 / 라운드 / a 슬라이더 / b 슬라이더
- **분홍 곡선**: 펜으로 그리는 발사 경로 미리보기 (a, b 바뀔 때마다 0.05초 안에 갱신)
- **풍선**: 라운드마다 5마리, 무작위 위치/색/크기

## 🎯 게임 흐름

1. ▶ 깃발 클릭 → 풍선 5마리가 화면에 등장
2. 분홍 곡선이 풍선을 통과하도록 a, b 조절
3. `Space` → 로켓이 곡선을 따라 비행 (접선 방향으로 회전)
4. 풍선과 충돌하면 풍선이 줄어들며 사라짐, 점수 +1
5. 5마리 모두 터뜨리면 라운드 +1, 새로운 풍선 5마리 자동 생성
6. 로켓이 땅(y < −160) 에 닿거나 화면 밖으로 나가면 사라지고 다음 발사 가능

## 🧮 수학 배경

발사체는 출발점 `(-200, -120)` 을 기준으로 다음 식을 따릅니다.

$$
y(x_{\text{step}}) = a \cdot x_{\text{step}}^2 + b \cdot x_{\text{step}}
$$

- **`a`** (이차항 계수)
  - `a < 0` → 위로 볼록한 포물선 (로켓이 솟구쳤다가 떨어짐)
  - `a > 0` → 아래로 볼록한 포물선
  - `|a|` 가 클수록 휘는 정도가 큼
- **`b`** (일차항 계수)
  - 출발점에서의 접선 기울기
  - `b = 0` → 수평으로 출발
  - `b > 0` → 위로 출발
- **로켓의 회전 각도** 는 미분값 `2a·x + b` (= 접선의 기울기)에서 `atan` 으로 계산됩니다.

> 왜 `c` 항(상수항)은 없냐? — 출발점이 `c` 역할을 합니다 (`-120` 부분). 일부러 단순화한 형태입니다.

## 🛠 내부 구조 (스프라이트 4개)

| 스프라이트 | 역할 | 주요 블록 |
|-----------|-----|----------|
| `Stage` | 직접 그린 하늘/구름/산/잔디 배경 | (코드 없음) |
| `로켓` 🚀 | 발사체 (Twemoji) | 화살표/스페이스 입력, 곡선 따라 이동, 접선 회전 |
| `풍선` 🎈 | 표적 (Twemoji, 클론 5개) | 충돌 감지, 풍선수 카운트, 라운드 갱신 |
| `곡선` | 보이지 않음, 펜으로 곡선 미리보기 + 수식 갱신 | `pen` 확장, 무한 루프 두 개 |

펜 확장(`pen`)을 사용합니다.

## 🔧 다시 빌드하기

```bash
python3 build.py
# → 다항함수_게임.sb3 갱신
```

`build.py` 안에서 바꿔보면 재미있는 것들:

- `BG_SVG` 의 색상/도형 → 배경 분위기 변경
- 화살표 키 변화량 (`0.002`, `0.10`) → 조작 민감도
- `repeat 70` → 발사 거리 (스텝 수)
- 풍선 라운드당 마리수 (`repeat 5`)
