# 탱크 배틀 · Tank Battle

탑다운 탱크 슈터. 방향키로 회전·전진하고 스페이스로 포탄을 쏴 적 탱크를 격파한다. 라운드마다 적이 늘고(최대 3) 적속도가 빨라진다. 엄폐물 뒤에 숨어 적 포탄을 피하자. 체력이 0이 되면 게임오버.

## 조작
- **↑/↓**: 전진 / 후진
- **←/→**: 회전
- **space**: 발사 (쿨다운 있음)

## 구현 메모
- 적/엄폐물/포탄은 클론으로 생성.
- **포탄은 두 스프라이트로 분리**: `포탄`(플레이어) / `적포탄`(적). 적 포탄을 별도 스프라이트로 둬서 발사 즉시 자기 탱크에 닿아 자폭하지 않게 하고, 종류 구분(전역 변수)을 없앴다.
- 적 격파: `적탱크` 가 `touching 포탄` → 점수+20, 잔여적-1, 클론 삭제.
- 플레이어 피해: `touching 적포탄 AND 무적=0` → 체력-1, 잠시 무적.
- 라운드 진행: 워처가 `잔여적=0` 을 기다렸다가 다음 라운드 방송.

## 검증 (headless scratch-vm + 좌표기반 touching 패치)
- 엄폐물 정확히 5개, 적 라운드별 정상 수(클론 증식 버그 수정 확인)
- 적 격파 시 점수/잔여적 정확, **잔여적 음수 없음** → 라운드 1→5 정상 진행
- 적 포탄이 날아가 플레이어 체력 감소
- 체력 0 → 게임상태=0 + 게임오버 배너 표시

### 수정한 버그 (이전엔 게임이 멈췄음)
1. **클론 증식**: `적탱크`/`엄폐물` 의 "클론 생성" 방송 핸들러가 클론에서도 실행돼 매 방송마다 기하급수로 늘어남(엄폐물 5→31, 적이 잔여적보다 많아짐). → `복제됨` 로컬 플래그로 원본만 생성하도록 가드.
2. **적 포탄 즉시 자폭 + 전역 종류 레이스**: 플레이어/적 포탄이 한 스프라이트라 적 포탄이 자기 탱크에 닿아 사라지고, 전역 `포탄종류` 가 엉켜 적이 자기 포탄에 죽어 **잔여적이 음수**가 됨 → 라운드 워처(`잔여적=0` 대기)가 영영 안 풀려 게임 정지. → 포탄 스프라이트 분리로 해결.
