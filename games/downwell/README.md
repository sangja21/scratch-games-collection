# 다운웰

> 건부츠를 신은 소년이 끝없이 깊은 우물로 **떨어진다.** 화면은 계속 위로 흘러(=내려감) 아래에서 적·가시·보석이 솟아오른다. 좌우로 피하고, **아래로 쏘면** 잠깐 떠올라(체공) 위기를 넘기지만 **탄약이 제한**이라 함부로 못 쏜다 — **땅이나 적을 밟으면(스톰프) 탄약이 가득 찬다.** 점프벌레는 밟아 죽이고, 박쥐와 가시고슴도치는 반드시 쏴야 한다. 땅에 안 닿고 연속 처치하면 **콤보 배수**로 점수가 곱으로 불어난다. 깊이 내려갈수록 강화 택1로 강해지고, 체력 0이면 GAME OVER. **내려간 깊이가 곧 점수다.**

인기 인디게임 **Downwell** 을 옮긴 세로 우물 낙하 슈팅 로그라이트다. 그리고 이 게임의 진짜 재미는 **"코드의 숫자를 직접 바꿔보며 내 게임으로 만드는 것"** 이다. 중력·부양력·탄약·적 체력 같은 모든 값이 한글 변수로 정리되어 있어서, 숫자 하나만 바꿔도 게임이 확 달라진다. 아래 [개조 손잡이](#️-개조-손잡이-바꿔보세요)와 [개조 미션 사다리](#-개조-미션-사다리)를 따라 하나씩 도전해 보자.

## 조작법

| 키 | 동작 |
|----|------|
| ← → | 좌우로 이동 (우물 안쪽 채널에서만, 위아래 위치는 카메라가 고정) |
| ↑ / Space | 점프 (착지 직후 잠깐 동안만 가능 — "코요테" 여유 시간) |
| ↓ / Z | **아래로 발사!** 쏠 때마다 탄약 −1, 그리고 잠깐 떠오른다(부양/체공) |
| 1 / 2 / 3 / 4 | 강화 깊이마다 뜨는 카드에서 강화 고르기 (`1` 탄약+ · `2` 연사+ · `3` 부양력+ · `4` 이동+) |

> **밟기(스톰프)와 충전**: 점프벌레나 발판 위에 떨어지면 **밟아서** 탄약이 가득 찬다. 탄약이 0이면 쏠 수도, 떠오를 수도 없으니 **밟아서 충전 → 다시 난사**가 핵심 리듬이다. 박쥐와 가시고슴도치는 밟으면 안 되고(찔린다!) 반드시 쏴야 한다.

## 게임 흐름

1. **시작** — 초록 깃발을 누르면 능력치가 기본값으로 맞춰지고, 소년이 우물 위쪽 카메라선(0, 40)에 선다. 곧 화면 아래에서 적·발판·보석이 솟아오르며 우물이 위로 흐르기 시작한다(= 내가 내려간다).
2. **낙하 & 사격** — ← → 로 좌우를 피하고, ↓ 또는 Z 로 아래로 쏜다. 쏠 때마다 "퓽" 소리와 함께 탄약이 1 줄고 낙하가 잠깐 느려지며 살짝 떠오른다(부양). **탄약이 다 떨어지면 더는 쏘지도 떠오르지도 못하니**, 점프벌레나 발판을 밟아("쿵·통") 탄약을 가득 채우자. 적은 색과 생김새로 구분한다 — 둥글고 초록인 **점프벌레**(밟기), 보라 날개의 **박쥐**(쏘기), 빨갛고 가시가 솟은 **가시고슴도치**(꼭 쏘기). 적을 맞히면 그 자리에 **데미지 숫자(`총알공격력`)가 통! 하고 떠오르고**, 적에 닿으면 체력이 1 깎이고 잠깐 무적이 된다.
3. **콤보 → 곱 점수** — **땅(발판)에 안 닿고** 공중에서 적을 연속으로 처치하면 `콤보`가 1씩 늘고, 처치할 때마다 점수가 `처치점수 × 콤보` 만큼 들어온다. 콤보가 오를수록 같은 적인데 점수가 **곱으로** 불어나고, 콤보음도 점점 높아진다. 단, **발판에 착지하면 콤보가 0으로 리셋**된다 — 통통 튀며 안 내려앉고 버티는 게 고득점의 비결이다. 처치한 적은 보석을 떨어뜨리고, 보석을 주우면 깊이(점수)에 보너스가 붙는다.
4. **강화로 강해짐** — 깊이가 `강화깊이`(1000)에 도달할 때마다 화면 가운데에 강화 카드가 뜬다. `1 탄약+`, `2 연사+`, `3 부양력+`, `4 이동+` 중 하나를 키로 고르면 낙하가 재개된다. 더 깊이 내려갈수록(`난이도깊이` 마다 `단계`가 1 올라) 박쥐·가시고슴도치가 섞여 나오고 적이 더 **자주** 쏟아진다.
5. **죽으면 처음부터** — 체력이 0이 되면 GAME OVER 배너가 뜨고 내려간 깊이(점수)가 보인다. 초록 깃발을 다시 누르면 능력치가 기본값으로 돌아가 우물 꼭대기부터 다시 시작한다.

게임을 바꾸는 모든 숫자는 **무대(Stage)의 초록 깃발 스크립트 한 곳**에 모여 있다. 코드 어디에도 숨겨진 숫자가 없으니, 여기 **32개** 손잡이만 돌리면 게임이 달라진다.

> 찾는 법: Scratch 에서 **무대**를 클릭 → 코드 탭 → 맨 위 `초록 깃발을 클릭했을 때` 묶음(노란 코멘트 "⚙️ 개조 손잡이"가 붙어 있다)을 보면 `중력 을(를) 0.55 로 정하기` 같은 블록들이 쭉 있다. 그 숫자를 바꾸면 끝!

## 🎯 개조 미션 사다리

쉬운 것부터 어려운 것까지. **하나씩 깨보면서 "코드를 망가뜨리고 고치는" 재미**를 느껴보자. 망가져도 괜찮다 — 초록 깃발만 다시 누르면 된다!

### 0층 — 숫자 하나 바꾸기 (무대 초록 깃발 스크립트)
- **둥둥 우주인**: `중력` 을 `0.55` → `0.2` 로. 천천히 떨어져 한참 떠 있다.
- **번개 추락**: `최대낙하` 를 `9` → `16` 으로. 쏜살같이 떨어진다(점수 빨리 쌓이지만 위험).
- **기관총 건부츠**: `연사간격` 을 `0.12` → `0.04` 로. 아래로 따발총처럼 쏟아지며 둥실 떠오른다.
- **탄창 가득**: `탄약최대` 를 `6` → `20` 으로. 공중에서 한참 난사하며 체공한다.
- **세진 총알**: `총알공격력` 을 `1` → `3` 으로. **적을 때려서 뜨는 데미지 숫자가 `1`→`3` 으로 바뀌는지** 직접 확인해 보자 — 데미지 팝업은 `총알공격력` 을 그대로 보여준다!
- **부양 슈퍼**: `부양력` 을 `1.6` → `3`, `부양상한` 을 `5` → `9` 로. 한 발 쏠 때마다 훅 떠오른다.
- **두부 멘탈**: `최대체력` 을 `4` → `1` 로. 한 대만 맞아도 위험! (난이도 급상승)
- **콤보 대박**: `처치점수` 를 `5` → `20` 으로. 콤보 ×N 의 점수 폭발이 훨씬 짜릿해진다.
- **지옥문**: `스폰간격` 을 `1.0` → `0.4` 로. 적이 두세 배로 솟아오른다.

### 1층 — 값을 식으로 묶기 (관계 만들기)
- **쏠수록 강한 반동**: 소년 발사에서 `change 낙하속도 by (0 - 부양력)` 을 `change 낙하속도 by (0 - (부양력 + (탄약 / 5)))` 로 — 탄약이 많을수록 한 발이 더 세게 떠오른다.
- **깊을수록 빠른 적**: 적 클론의 흔들 식에서 `내속도` 대신 `(내속도 + (단계 / 3))` 로 — 깊이 갈수록 적이 더 빠르게 흔들린다.
- **콤보가 곧 화력**: 적의 총알 피격에서 `change 내체력 by (0 - 총알공격력)` 을 `change 내체력 by (0 - (총알공격력 + (콤보 / 3)))` 로 — 콤보가 높을수록 총알도 세진다.

### 2층 — if 조건 더하기
- **막판 회복**: 소년의 발판 착지 처리에 `if 체력 < 최대체력 : change 체력 by 1` 을 추가 — 착지할 때마다 체력 1 회복(쉬운 모드).
- **콤보 무적**: 적 처치(콤보+1) 직후에 `if 콤보 >= 5 : set 무적 = 무적시간` 추가 — 5콤보를 넘으면 잠깐 무적으로 폭주한다.
- **바닥 가시 함정**: 적 클론에 `if (단계 >= 3) and (적종류 = 1) : set 스톰프가능 = 0` 을 넣어, 후반엔 점프벌레도 못 밟게(난이도↑).

### 3층 — 규칙 자체를 바꾸기 (분기)
- **강화 5번 추가(점프력+)**: `강화카드` 에 `if key [5] : change 점프력 by 강화량` 선택지를 더하고, 카드 그림에 5번 칸을 그려 넣는다.
- **강화 6번 추가(공격력+/최대체력+)**: `if key [6] : change 총알공격력 by 강화량`(또는 `change 최대체력 by 강화량 ; change 체력 by 강화량`)를 추가.
- **랜덤 강화**: 강화 선택을 키 입력 대신 `1부터 4 사이의 난수` 로 자동 결정하게 바꿔, 운에 맡겨보자.

### 4층 — 새 메커닉 만들기 (구조)
- **3연발 산탄 건부츠**: `총알` 의 발사를 `repeat 3 : create clone` 으로 바꾸고, 클론마다 x 를 조금씩 벌려(`-12 / 0 / +12`) 아래로 부채꼴 산탄을 만든다.
- **회복 보석 추가**: `보석` 을 복제해 `체력보석` 스프라이트를 만들고, 적 처치 시 가끔(`1부터 5 사이의 난수 = 1`) 떨어뜨려 닿으면 `change 체력 by 1`.
- **분열 점프벌레**: 적 처치 시 `if 적종류 = 1 : (작은 점프벌레 2마리 스폰)` 을 넣어, 약한 적이 죽으며 둘로 쪼개지게(스폰 채널 재사용).
- **콤보 화면 연출**: 콤보가 오를 때 `데미지표시값` 에 `콤보` 를 넣어 큰 숫자를 띄우거나, 콤보음 pitch 식(`콤보 × 25`)을 바꿔 더 짜릿하게.

## 학습 포인트 (부가)

**메인은 어디까지나 빠른 액션**이다 — 막 누르고 쏘며 통통 튀는 손맛이 먼저다. 그 위에 "코딩 학습"이 얹혀 있다: 모든 능력치가 한글 튜닝 변수(60개 중 **32개가 개조 손잡이**)로 노출되어, 아이가 (0층) 숫자를 바꾸고 → (1층) 값을 식으로 묶고 → (2층) if 조건을 더하고 → (4층) 새 스프라이트를 추가하며 **변수·연산·조건·클론·방송·좌표/스크롤** 을 게임을 망가뜨리고 고치며 체득한다. 무대를 클릭하고 코드 탭을 열면 곳곳에 붙은 **노란 코멘트(가이드 투어)** 가 각 부분이 무엇을 하는지 바로 알려준다.

그리고 액션의 *결과로* 수학이 슬쩍 드러난다:
- **✖️ 곱셈 (콤보 배수)** — 땅에 안 닿고 연속 처치하면 `콤보`가 1씩 늘고 점수 = `처치점수 × 콤보`. 콤보가 오를수록 같은 적인데 점수가 **곱으로** 불어나고, 콤보음도 같이 높아진다 → "곱하기는 더하기를 빠르게 한 것"을 손과 귀로 만난다.
- **🧭 좌표·부호 (스크롤)** — `낙하속도`가 양수면 하강, 음수면 부양. 같은 변수의 부호 하나로 화면 전체가 위로 또는 아래로 흐르는 걸 보며 양수·음수와 방향을 체감한다.
- **💰 탄약 경제** — 쏘면 탄약이 줄고(−1) 밟으면 가득 찬다. "쓴 만큼 줄고, 채운 만큼 다시 쏠 수 있다"는 자원 관리(증가·감소·상한)를 몸으로 익힌다.
- 그 밖에 **누적**(깊이 = 낙하속도의 합), **확률적 스폰**(난수로 x·종류 결정), **점점 세지는 난이도 곡선**(단계 = 깊이 ÷ 난이도깊이).
