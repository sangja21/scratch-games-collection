# 캐슬 디펜스

> 가장자리에서 S자 길을 따라 성으로 행진하는 고블린·오크·트롤 무리를, 길 옆 잔디에 **포탑을 지어** 막아내는 클래식 로그라이트 타워디펜스. 포탑은 **사거리 안 가장 가까운 몬스터에게 알아서 발사**하니, 너는 *어디에 무슨 포탑을 지을지*만 정하면 된다. 몬스터를 잡으면 골드가 쌓이고, 골드로 포탑을 더 짓는다. 한 웨이브를 막아내면 강화 카드 넷 중 하나를 `1·2·3·4` 키로 골라 점점 강해진다. 웨이브가 오를수록 몬스터가 더 많이·세게·빠르게 몰려오고, 성체력이 0이 되면 GAME OVER. **도달한 웨이브가 곧 점수다.**

이 게임의 진짜 재미는 **"코드의 숫자를 직접 바꿔보며 내 게임으로 만드는 것"** 이다. 포탑 사거리·공격력·연사, 몬스터 체력·속도, 골드 경제, 심지어 **길 좌표까지** 모든 값이 한글 변수로 정리되어 있어서, 숫자 하나만 바꿔도 게임이 확 달라진다. 아래 [개조 손잡이](#️-개조-손잡이-바꿔보세요)와 [개조 미션 사다리](#-개조-미션-사다리)를 따라 하나씩 도전해 보자.

## 조작법

| 입력 | 동작 |
|------|------|
| 하단 팔레트 버튼 클릭 | 지을 포탑 고르기 (`1` 화살탑 · `2` 대포탑🔒 · `3` 마법탑🔒) — 해금된 것만. 고른 버튼은 **노란 테두리**로 강조된다 |
| 팔레트 `4` 성수리 버튼 클릭 | **즉시** 골드 `수리비용`을 내고 성체력을 `수리량`만큼 회복 (성최대체력에서 멈춤). 골드 부족·풀피면 에러음만 — 포탑 선택은 안 바뀜 |
| 잔디 위 클릭 | 고른 포탑 설치 (길 위·성 위·다른 포탑 위는 금지, 골드가 충분해야 설치) |
| **`스페이스`** 키 (또는 좌상단 ⚡ 버튼 클릭) | **전체 번개 주문** — 화면의 **모든 몬스터**에게 `주문공격력`만큼 동시에 번개 피해! '쩍-쾅!' 천둥과 함께 화면이 두 번 번쩍이고 위에서 **지그재그 번개 줄기**가 내리꽂히며, 맞은 몬스터가 흰색으로 번쩍인다. 쓰고 나면 `주문쿨`초 동안 재충전(그동안 버튼이 *충전중*, 준비되면 *준비!*) |
| (자동) | 설치한 포탑이 사거리 안 가장 가까운 몬스터에게 알아서 발사 |
| `1` / `2` / `3` / `4` | 웨이브 클리어 후 강화 고르기 (`1` 공격력+ · `2` 사거리+ · `3` 연사+ · `4` 골드+) |
| 초록 깃발 | 시작 / 다시 시작 (능력치·웨이브 기본값으로 리셋) |

> 포탑을 고르면 **반투명 유령 미리보기**가 그 포탑 모습으로 마우스를 따라다녀, 지금 무엇을 어디에 세우는지 한눈에 보인다(시각 전용 — 설치 가능 여부는 가운데 작은 십자선 커서가 따로 판정해 설치가 쉽다). **초록**이면 설치 가능, **빨강**이면 불가(길·성·팔레트·다른 포탑 위거나 골드 부족). 좌상단 HUD 에 `웨이브`(점수)·`골드`·`성체력` 3종이 보인다.

## 게임 흐름

1. **시작** — 초록 깃발을 누르면 모든 능력치가 기본값으로 맞춰지고, S자 돌길과 성이 그려진다. 곧 왼쪽 가장자리에서 고블린이 행진을 시작한다(웨이브 시작마다 묵직한 뿔피리 소리).
2. **설치 & 방어** — 하단 팔레트에서 포탑을 고르면(고른 버튼이 노란 테두리로 강조되고 **반투명 유령 미리보기**가 마우스를 따라온다) 잔디를 클릭해 설치한다. 화살탑은 **싸고 빠르고**(단일 명중), 대포탑은 **느리지만 광역**(무리를 한 방에), 마법탑은 **사거리 길고 강하다**. 포탑은 사거리 안 가장 가까운 몬스터를 자동으로 조준해 타입별 발사음과 함께 발사하고, 맞은 자리에 **데미지 숫자가 흰색으로 통! 떠오른다.** 성이 위험할 땐 팔레트 맨 오른쪽 **`4` 성수리** 버튼으로 골드를 내고 성체력을 즉시 회복할 수 있다.
3. **처치 → 골드** — 몬스터를 잡으면 동전 소리와 함께 **금색 +골드 숫자**가 떠오르고 골드가 쌓인다. 그 골드로 포탑을 더 짓는다. 고블린(약·빠름)·오크(중)·트롤(강·단단)은 색과 크기로 구분된다. 적이 우르르 몰릴 땐 **`스페이스` 전체 번개 주문**으로 화면의 모든 몬스터를 한 번에 지져 위기를 넘긴다(좌상단 ⚡ 버튼이 *준비!*면 사용 가능, *충전중*이면 대기 — `주문쿨`초마다 다시 충전).
4. **웨이브 클리어 → 강화** — 이번 웨이브 몬스터를 모두 막으면 골드 보너스를 받고 화면 가운데에 강화 카드가 뜬다. `1 공격력+`, `2 사거리+`, `3 연사+`, `4 골드+` 중 하나를 키로 고르면 **곧바로** 다음 웨이브가 시작된다(대기 없음). 웨이브 2에 대포탑, 웨이브 4에 마법탑이 해금된다.
5. **점점 어려워짐** — 웨이브가 오를수록 몬스터 수·체력·속도가 비례해 늘고, 등장 종류도 *고블린만 → 고블린·오크 → 셋 섞임* 으로 거칠어진다. 성에 몬스터가 닿을 때마다 경보음과 함께 `성체력`이 1씩 깎인다.
6. **죽으면 처음부터** — 성체력이 0이 되면 GAME OVER 배너가 뜨고 도달한 웨이브(점수)가 보인다. 초록 깃발을 다시 누르면 처음부터 다시 시작한다.

> 찾는 법: Scratch 에서 **무대**를 클릭 → 코드 탭 → 맨 위 `초록 깃발을 클릭했을 때` 묶음을 보면 `기본골드 을(를) 150 로 정하기` 같은 블록이 쭉 있다. 그 숫자만 바꾸면 끝! (노란 코멘트 "🛠️ 개조 손잡이" 가 붙어 있다.)

## 🎯 개조 미션 사다리

쉬운 것부터 어려운 것까지. **하나씩 깨보면서 "코드를 망가뜨리고 고치는" 재미**를 느껴보자. 망가져도 괜찮다 — 초록 깃발만 다시 누르면 된다!

### 0층 — 숫자 하나 바꾸기 (무대 초록 깃발 스크립트)
- **부자 영주**: `기본골드` 를 `150` → `500` 으로. 시작부터 포탑을 잔뜩 지을 수 있다.
- **싸진 화살탑**: `화살탑가격` 을 `50` → `10` 으로. 화살탑을 길에 도배!
- **빠른 손**: `화살탑_간격` 을 `0.45` → `0.15` 로. 화살이 따발총처럼 쏟아진다.
- **묵직한 대포**: `대포탑_폭발반경` 을 `60` → `120` 으로. 한 방에 무리를 쓸어버린다.
- **세진 마법탑**: `마법탑_공격력` 을 `4` → `10` 으로. **맞을 때 뜨는 데미지 숫자가 바뀌는지** 직접 확인!
- **거북이 트롤**: `트롤_속도` 를 `0.9` → `0.3` 으로. 트롤이 더 느릿느릿 다가온다.
- **튼튼한 성**: `성최대체력` 을 `20` → `60` 으로. 훨씬 많이 뚫려도 버틴다.
- **물량 지옥**: `몬스터간격` 을 `0.8` → `0.2` 로. 몬스터가 우르르 몰려온다(난이도 급상승).
- **포탑 빨리 해금**: `대포탑해금웨이브`·`마법탑해금웨이브` 를 `2`,`4` → `1`,`1` 로. 처음부터 세 포탑을 다 쓴다!

### 1층 — 값을 식으로 묶기 (관계 만들기)
- **웨이브 비례 보상**: 무대 클리어 보너스를 `웨이브클리어골드` 대신 `웨이브클리어골드 × 웨이브` 로 — 후반일수록 골드가 팍팍 들어온다.
- **점점 단단해지는 적**: `웨이브체력증가` 를 `2` → `5` 로. 웨이브가 오를수록 적 체력이 가파르게 오른다 (실제 체력 = 종류체력 + (웨이브−1) × 이 값).
- **연사 강화 더 세게**: 강화카드 `3` 번을 `연사보너스 × 0.85` → `× 0.7` 로. 한 번 고를 때마다 연사가 훨씬 빨라진다.

### 2층 — if 조건 더하기
- **보스 웨이브**: 무대 웨이브 매니저에 `만약 (웨이브 mod 5 = 0) 이면 → 생성타입 을 3 으로 정하기` 를 추가 — 5의 배수 웨이브는 트롤만 몰려오는 보스전.
- **위기 보너스**: 무대 감시 스크립트에 `만약 성체력 = 1 이면 → 골드 을 100 만큼 바꾸기` 를 더해, 막판에 역전 자금을 준다.
- **강화 5번 추가(성 수리)**: 강화카드에 `만약 5키 누름 → 성체력 을 5 만큼 바꾸기` 선택지를 더하고 카드에 5번 칸을 그린다.

### 3층 — 규칙 자체를 바꾸기 (분기)
- **랜덤 강화**: 강화 선택을 키 입력 대신 `1부터 4 사이의 난수` 로 자동 결정하게 — 운에 맡기기.
- **마법탑 즉시 해금 카드**: 강화카드에 `만약 5키 누름 → 마법탑해금 을 1 로 정하기` 를 더해, 강화로 직접 포탑을 해금.
- **분열 고블린**: 몬스터 처치 시 `만약 내타입 = 1 이면 → 같은 경로 다음 점에 작은 고블린 2마리 스폰` 을 넣어, 약한 적이 죽으며 둘로 쪼개지게(스폰 채널 재사용).

### 4층 — 새 메커닉 만들기 (구조)
- **길 바꾸기**: 무대의 `경로X`·`경로Y` 리스트 좌표를 바꾸거나 점을 추가해 더 구불구불한 길을 만들어 보자. 몬스터가 새 길로 행진한다!
- **느려지는 마법탑**: 마법탑 `타격` 에 닿은 몬스터의 `내속도` 를 잠시 줄이는 슬로우 효과(새 클론-로컬 변수)를 추가.
- **황금 몬스터**: 가끔(`1부터 10 사이의 난수 = 1`) 스폰되는 반짝이는 몬스터를 만들어, 잡으면 골드 5배(새 타입/색).
- **화려한 골드 숫자**: 숫자팝업 골드 클론의 떠오르는 속도·페이드를 키우거나 표시값에 `× 2` 를 곱해 두 배로 연출.

## 학습 포인트 (부가)

**"코딩 학습"이 핵심 가치**다 — 모든 능력치(골드·가격·포탑 사거리/공격력/간격·몬스터 체력/속도·웨이브 스케일·길 좌표)가 한글 튜닝 변수·리스트로 노출되어, 아이가 (0층) 숫자를 바꾸고 → (1층) 값을 식으로 묶고 → (2층) if 조건을 더하고 → (4층) 길/새 포탑을 추가하며, **변수·리스트·연산·조건·클론·broadcast and wait** 라는 프로그래밍 개념을 게임을 망가뜨리고 고치며 자연스럽게 체득한다. 코드 곳곳에 **노란 코멘트(가이드 투어)** 가 붙어 있어, 무대나 스프라이트를 클릭하고 코드 탭을 열면 각 부분이 무엇을 하는지 바로 읽을 수 있다.

그리고 액션의 *결과로* 수학·알고리즘이 슬쩍 드러난다:
- **🗺️ 경로 / 인덱스** — 몬스터는 `경로X`·`경로Y` 리스트의 6개 길목을 `현재점`(1, 2, 3…)으로 하나씩 따라간다. 리스트와 인덱스로 "정해진 길 따라가기"를 만난다. (좌표만 바꾸면 길이 바뀐다!)
- **🎯 최솟값 찾기** — 자동 조준은 `조준요청 방송하고 기다리기` 한 줄. 모든 몬스터가 차례로 "내가 사거리 안에서 제일 가까운가?"를 검사해 가장 가까운 적(거리의 최솟값)을 경쟁 없이 찾는다.
- **💰 자원 관리(골드 수지)** — 처치·클리어로 들어오는 골드와 설치·강화로 나가는 골드의 균형. "벌이 > 지출"을 유지해야 포탑을 늘릴 수 있다.
- **📈 증가 곡선(웨이브 스케일)** — 몬스터 수·체력·속도가 `기본 + (웨이브−1) × 증가량` 으로 자란다. 웨이브가 오를수록 난이도가 어떻게 가팔라지는지 숫자로 만난다.

> 어디까지나 직관적인 타워디펜스로 포장된 채, 빠른 설치·자동 발사·강화 택1의 **전략 액션**이 메인이다.
