# 🌋 리히터 진앙 추적

두 도시(A, B) 의 지진계에 진폭이 표시됩니다. **슬라이더로 두 지진의 매그니튜드 차 ΔM 을 추측해** ±0.3 이내로 맞추면 정답입니다. 60초 안에 최대한 많이 통과시키세요.

> **학습 포인트** — 리히터 매그니튜드는 로그 스케일이라 **매그니튜드 1 차이 = 진폭 10 배 차이** 입니다. 규모 6 과 규모 9 지진은 단순히 "1.5 배 큰 것" 이 아니라 **진폭이 1000 배 차이** 납니다. 슬라이더로 직접 맞춰 보며 이 사실을 손에 익힙니다.

## 🎮 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 학습 게임 / 슬라이더 퀴즈 |
| 핵심 메커닉 | 두 지진의 진폭으로 ΔM 추측 |
| 학습 주제 | `M = log₁₀(A/A₀)`, 로그의 차 = 비의 로그 |
| 제한 시간 | 60초 |
| 권장 학년 | 고등학교 1학년 (지수·로그 단원) + 지구과학 |

## 🕹 조작법

| 입력 | 동작 |
|------|------|
| `내답ΔM` 슬라이더 (좌측 하단) | 매그니튜드 차 추측값 입력 (0.0 ~ 5.0, 연속) |
| `Space` | 시도 — 차이가 0.3 이내면 정답 (+1점, 새 라운드) |

## 🎯 라운드 / 시나리오

| 라운드 | 진폭 A | 진폭 B | 정답 ΔM | 비율 | 학습 의도 |
|--------|--------|--------|---------|------|----------|
| 1 | 100 | 10 | 1.0 | 10:1 | 매그니튜드 1 = 진폭 10배 |
| 2 | 10,000 | 100 | 2.0 | 100:1 | 매그니튜드 2 = 진폭 100배 |
| 3 | 100,000 | 100 | 3.0 | 1000:1 | 매그니튜드 3 = 진폭 1000배 |
| 4 | 10,000 | 10 | 3.0 | 1000:1 | 같은 ΔM, 다른 절대값 |
| 5 | 100,000 | 10 | 4.0 | 10000:1 | 가장 큰 차이 (학습 정점) |
| 6+ | 10^랜덤 | base | 1~4 | 10/100/1000/10000 | 자유 라운드 |

기준 진폭 `A₀ = 1` 으로 고정.

## 🧮 학습 포인트 — 진폭 10배 = 매그니튜드 +1

### 1. 매그니튜드의 정의 (리히터)

$$ M = \log_{10}\!\left(\dfrac{A}{A_0}\right) $$

- 진폭(A)이 10배 → M 은 +1
- 진폭이 100배 → M 은 +2
- 진폭이 1000배 → M 은 +3

### 2. 두 지진의 매그니튜드 차

$$ \Delta M = M_A - M_B = \log_{10}\!\left(\dfrac{A_A}{A_0}\right) - \log_{10}\!\left(\dfrac{A_B}{A_0}\right) = \log_{10}\!\left(\dfrac{A_A}{A_B}\right) $$

기준 진폭 A₀ 가 **소거** 되어, **두 진폭의 비 만으로 ΔM 이 결정** 됩니다. 이게 로그의 핵심 성질 — `log a - log b = log(a/b)`.

| 진폭 비 | ΔM |
|---------|-----|
| 10 : 1 | 1.0 |
| 100 : 1 | 2.0 |
| 1000 : 1 | 3.0 |
| 10000 : 1 | 4.0 |

### 3. 로그 스케일의 시각화

이 게임의 진폭 막대는 **`size = 50·log₁₀(진폭) + 30`** 으로 그려집니다. 그래서:
- 진폭 1 → size 30 (가장 작음)
- 진폭 100 → size 130
- 진폭 100000 → size 280

진폭이 10000 배 차이 나도 막대 길이 차이는 4 배 정도. **로그 스케일이 큰 비율을 한 화면에 담는 방법** 임을 시각으로 보여줍니다.

### 4. Scratch 의 `log` 블록

이 게임은 `operator_mathop` 블록의 `OPERATOR: "log"` 를 5 번 사용합니다 (Stage 에 3 번 — 매그A·매그B·정답ΔM 계산, 진폭바 sprite 2 개에 각각 1 번씩 — size 갱신). Scratch 의 `log` 는 **상용로그(밑 10)** 라서 리히터 정의에 그대로 맞습니다.

## 🧪 권장 학습 흐름

1. **라운드 1**: A=100, B=10. ΔM 슬라이더를 **1.0** 으로 맞추세요. 진폭이 10배 차이 = 매그니튜드 +1.
2. **라운드 2**: A=10000, B=100. ΔM = **2.0**. 진폭 100배 = 매그니튜드 +2.
3. **라운드 3·4**: 진폭 1000배 → ΔM = 3.0. 절대값이 달라도 비율이 같으면 ΔM 도 같음을 확인.
4. **라운드 5**: 진폭 10000배 → ΔM = 4.0. 실제 큰 지진과 작은 지진의 차이 정도.
5. **라운드 6+**: 무작위로 1~4 의 정수 ΔM 이 나옴. 머릿속으로 `log₁₀(비율)` 을 빠르게 계산해 보세요.
6. 막힐 때 `힌트 ▼` 버튼 — `정답: X.X ΔM = 진폭 N 배` 가 표시됩니다.
