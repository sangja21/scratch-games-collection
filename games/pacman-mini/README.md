# 팩맨 미니 (Pac-Man Mini)

방향키로 팩맨을 움직여 15열×11행 미로의 도트(점)를 전부 먹는다. 유령 4마리가 미로를 돌아다니며 쫓아오고, 닿으면 라이프가 하나 줄어든다(라이프 3). 네 모서리 안쪽의 큰 파워펠릿을 먹으면 7초 동안 유령들이 파랗게 겁먹어, 거꾸로 유령을 잡아먹을 수 있다(잡힌 유령은 집으로 돌아간다). 도트와 펠릿 92개를 모두 먹으면 승리. 1980년 남코 "Pac-Man"의 클래식 메커닉을 초등학생용으로 간소화한 버전이다.

## 조작

| 키 | 동작 |
|----|------|
| → | 오른쪽으로 진행 |
| ↑ | 위로 진행 |
| ← | 왼쪽으로 진행 |
| ↓ | 아래로 진행 |

- 팩맨은 매 스텝(0.16초)마다 진행 방향으로 한 칸(24px)씩 자동으로 움직입니다. 방향키는 "다음에 꺾을 방향"을 예약합니다.
- 예약한 방향이 벽이면 무시되고 기존 방향을 유지합니다 — 교차로에 닿기 전에 미리 키를 눌러두면 그 자리에서 자동으로 꺾입니다.
- 스네이크와 달리 정반대 방향으로도 즉시 돌 수 있습니다(역방향 차단 없음).
- 미로 좌우 가운데(row5)에는 터널이 있어, 한쪽 끝으로 나가면 반대쪽 끝으로 워프합니다. 팩맨과 유령 모두 사용합니다.

## 게임 흐름

1. 초록 깃발 ▶ 을 누르면 미로가 그려지고 도트·펠릿 92개가 배치됩니다. "READY!" 배너가 잠깐 뜨고 팩맨은 미로 중앙 위쪽(7,3)에서, 유령 4마리는 가운데 유령집에서 출발합니다.
2. 방향키로 도트를 먹으며 미로를 돌아다닙니다. 도트 +10점, 파워펠릿 +50점.
3. 정상 유령에 닿으면 라이프 -1, 팩맨과 유령이 시작 위치로 돌아가고 잠깐 "READY!" 후 재개됩니다. (이미 먹은 도트는 그대로 유지)
4. 파워펠릿을 먹으면 7초간 유령들이 파랗게 겁먹습니다. 이때 유령에 닿으면 +200점, 그 유령은 눈만 남아 집으로 복귀했다가 다시 정상이 됩니다.
5. 도트·펠릿을 전부 먹으면 "YOU WIN!", 라이프가 0이 되면 "GAME OVER" 배너가 뜹니다.
6. 다시 깃발 ▶ 을 누르면 새 게임이 시작됩니다.

## 룰

- **점수**: 도트 +10, 파워펠릿 +50, 겁먹은 유령 잡기 +200.
- **라이프 3**: 정상 유령에 닿을 때마다 -1, 0이 되면 게임오버.
- **승리**: 미로의 먹이(도트 88 + 펠릿 4 = 92개)를 모두 먹으면 승리.
- **유령 4마리 성향이 다릅니다** — 빨강은 가장 공격적(70% 추격), 분홍 50%, 청록 30%, 주황은 거의 안 쫓아오고 어슬렁거립니다(완전 랜덤). 추격이 아닌 순간엔 무작위로 갈림길을 고릅니다.
- 유령은 왔던 길로 곧장 되돌지 않아(뒤로 돌기 금지) 자연스럽게 미로를 순찰합니다.
- 최고기록은 점수가 갱신될 때 함께 기록됩니다.

## 작동 원리 (내부 구조)

추상적인 학습 콘셉트는 없는 순수 액션 게임이지만, 미로를 Scratch에서 다루는 방식이 핵심 설계 포인트입니다.

- **미로는 1차원 리스트**: 15열×11행 격자를 길이 165의 `보드` 리스트로 평탄화합니다. 칸 (열 col, 행 row)은 인덱스 `idx = row*15 + col + 1` 로 읽고 씁니다. 값은 0=벽, 1=도트, 2=파워펠릿, 3=빈통로입니다.
- **좌표 변환**: 칸 중심의 무대 좌표는 `X = -168 + col*24`, `Y = 120 - row*24`. 팩맨·유령은 항상 칸 중심에 위치하며 한 번에 24px씩만 이동해 격자에 딱 맞습니다.
- **벽 판정의 단일 진실원**: 다음 칸으로 가기 전에 `item(다음idx) of 보드 = 0` 인지만 검사합니다. 벽이면 이동하지 않고 멈춥니다. 벽(0) 값은 게임 내내 변하지 않습니다.
- **먹기**: 도착한 칸이 도트/펠릿이면 보드 값을 3(빈통로)으로 치환하고 점수·남은먹이를 갱신합니다. 미로 배경의 벽은 클론이 아니라 배경 SVG에 직접 그려 클론을 절약하고, 도트·펠릿만 클론(최대 92개)으로 렌더합니다.
- **유령 AI (교차로 확률 추격)**: 각 유령은 매 스텝 갈 수 있는 방향(벽 아님 + 뒤로 돌기 제외)을 모은 뒤, 유령번호별 추격 확률로 "팩맨과의 맨해튼 거리를 가장 줄이는 방향"을 고르거나(추격) 후보 중 무작위로 고릅니다(순찰).

## 실행

1. `팩맨미니.sb3` 다운로드
2. [scratch.mit.edu](https://scratch.mit.edu) → 만들기 → 파일 → 컴퓨터에서 가져오기
3. 초록 깃발 ▶ 클릭

## 다시 빌드

```bash
cd games/pacman-mini
python3 build.py
# → 팩맨미니.sb3 갱신
```

`팩맨미니.sb3` 파일이 생성됩니다. Scratch "내 작업실"에 업로드하거나 Scratch 데스크탑에서 열어주세요.
