# 우주정거장 도킹 (station-docking)

> 우주선을 4방향 분사로 정밀하게 움직여 우주정거장의 도킹 포트에 살포시 닿아야 하는 정밀 조작 게임. 너무 빠르게 닿으면 충돌!

## 조작

- **위/아래/왼/오 화살표** — 해당 방향으로 분사 (관성 누적). 분사할 때마다 연료 -1.
- 회전은 없음. 우주선은 항상 같은 방향. 분사로만 움직임.

## 규칙

- 우주는 마찰이 없어요. 한번 움직이기 시작하면 계속 미끄러집니다.
- 우주정거장의 노란색 도킹 포트(왼쪽)에 우주선이 닿으면 됩니다.
- **천천히 닿아야** 도킹 성공! (속도 `|VX| < 1.5` 그리고 `|VY| < 1.5`)
- **너무 빠르게 닿으면** 충돌 → 게임오버.
- 연료가 0 이 되고 우주선도 멈추면 게임오버.

## 점수

- 도킹 성공 시: 점수 += 100 + 남은연료
- 라운드 +1 마다 우주정거장 위치(y)가 무작위로 변경

## 빌드

```sh
python3 build.py
# → 우주정거장_도킹.sb3
```

Scratch 3.0 또는 https://scratch.mit.edu 에 업로드해서 플레이.

## 구조 요약

- 5 스프라이트 (Stage / 우주선 / 정거장 / 도킹성공 / 게임오버)
- 7 변수, 4 broadcasts, 204 블록
- 베이스: `asteroids` 의 관성·분사 패턴 (회전 제거, 4방향 분기로 단순화)
