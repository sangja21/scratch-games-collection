# 👾 외계인 침공 (Alien Invasion)

갤러그 / 스페이스 인베이더 스타일의 클래식 슈팅 게임. 우주선을 조종해 외계인 격자를 다 격파하면 다음 라운드, 라이프 0이 되거나 외계인 격자가 바닥에 닿으면 게임오버.

> **컬렉션 안에서의 위치** — 다른 게임들이 학습형이라면, 이 게임은 *순수 액션 보너스*. 컨셉 일관성보다 슈팅 게임의 재미를 우선했어요.

## 🎮 게임 개요

| 항목 | 내용 |
|------|------|
| 장르 | 슈팅 (고정형 슈팅 / fixed shooter) |
| 핵심 메커닉 | 좌우 이동 + 발사, 외계인 격자가 점점 가까워짐 |
| 영감 | Galaga (1981), Space Invaders (1978) |
| 플레이 시간 | 라이프 / 라운드에 따라 1~10분 |

## 🕹 조작법

| 키 | 동작 |
|---|------|
| `←` / `→` | 우주선 좌우 이동 |
| `Space` | 총알 발사 |

## 🎯 점수 / 라이프 규칙

| 상황 | 결과 |
|------|------|
| 외계인 1마리 격파 | **점수 +10** |
| 모든 외계인 격파 | **라운드 +1, 적 이동속도 +0.5** |
| 적 총알에 맞음 | **라이프 −1** |
| 라이프 0 또는 외계인 격자가 바닥선(y < −130) 도달 | **게임 오버** |

## 🎯 게임 흐름

1. ▶ 깃발 클릭 → 점수 0, 라이프 3, 라운드 1, 적_속도 1.5 로 초기화
2. 외계인 5×3 격자(15마리)가 0.02초 간격으로 차례로 등장
3. 격자 전체가 좌우로 움직임. 진영X 가 ±80 도달 시 한 줄 내려옴 (15px), 방향 반전
4. 외계인은 각자 매 틱 1/400 확률로 자기 위치에서 적 총알 발사 (15마리 ⇒ 약 1초당 1발 안팎)
5. 플레이어 총알이 외계인에 닿음 → 점수 +10, 적수 −1, 외계인 즉시 사라짐 (높은 pop 사운드)
6. 적 총알이 우주선에 닿음 → 라이프 −1, 적 총알 사라짐 (낮은 pop 사운드)
7. 적수가 0이 되면 자동으로 다음 라운드 (적_속도 += 0.5, 격자 위치 리셋, 다시 15마리 스폰)

## 🛠 내부 구조 (스프라이트 5개)

| 스프라이트 | 역할 | 핵심 코드 |
|-----------|------|----------|
| `Stage` | 우주 배경, 게임 상태/변수, **격자 이동 루프** | `진영X` 가 ±80 도달 시 방향 전환 + 한 줄 내려옴, `진영Y < -130` 이면 게임오버 |
| `우주선` 🚀 | 플레이어. ←→ 이동, Space 발사 | `repeat-until 게임상태=0` 루프 안에서 키 상태 체크 |
| `총알` | 플레이어 총알 (클론) | 위로 13px씩, `외계인` 닿으면 자기 삭제 |
| `외계인` 👾 | 적. 클론 15마리 | `(진영X+자기X, 진영Y+자기Y)` 로 매 틱 이동, 총알 닿으면 점수+10 후 자기 삭제, 1/400 확률로 적 총알 발사 |
| `적탄` | 적 총알 (클론) | 아래로 7px씩, `우주선` 닿으면 라이프 -1 후 자기 삭제 |

### 격자 이동 메커니즘 (포인트)

각 외계인이 독립적으로 가장자리를 검사하면 클론끼리 동기화 문제가 생깁니다. 그래서 **stage 의 `진영X`, `진영Y`** 를 단일 진실 원점으로 두고, 각 외계인 클론은 자기 슬롯 오프셋 `자기X`, `자기Y` 만 가져요. 매 틱 위치 = `(진영X + 자기X, 진영Y + 자기Y)`.

```
Stage 격자 이동 루프:
  repeat until 게임상태=0:
    진영X += 적_방향 × 적_속도
    if 진영X > 80:  적_방향 = -1, 진영Y -= 15, 진영X = 80
    if 진영X < -80: 적_방향 =  1, 진영Y -= 15, 진영X = -80
    if 진영Y < -130: 게임상태 = 0   # 격자가 바닥선에 닿음
    wait 0.04
```

이 방식이라 클론이 몇 마리든 격자 전체가 한 몸처럼 움직입니다.

## 🔧 다시 빌드하기

```bash
python3 build.py
# → 외계인_침공.sb3 갱신
```

조절해볼 만한 값:
- `적_속도` 초기값 1.5 → 더 어렵게 / 쉽게
- `1/400` 발사 확률 → 적 공격 빈도
- 격자 행/열 (`cols = [-100, -50, 0, 50, 100]`, `rows = [0, -38, -76]`) → 외계인 수
- `진영Y < -130` → 격자 도달선
