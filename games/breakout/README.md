# 벽돌깨기 (Breakout)

화면 하단의 패들을 좌/우 화살표로 움직여 공을 받아친다. 공은 좌·우·위 3면 벽에서 튕기고, 패들에 맞으면 항상 위로 반사된다. 패들 중심에서 얼마나 벗어나 맞느냐(오프셋)에 따라 공의 좌우 각도가 바뀌어 조준할 수 있다. 위쪽에는 8열 × 4행 = 32개의 벽돌 격자가 있고, 공이 벽돌에 닿으면 그 벽돌이 깨지며 점수를 얻고 공의 진행 방향(Y)이 반전된다. 윗행일수록 점수가 높다. 벽돌을 모두 깨면 다음 라운드(공이 더 빨라지고 격자 리셋). 공을 못 받아 바닥으로 떨어뜨리면 목숨 -1, 목숨 3개가 모두 닳으면 게임오버. 1976 아타리 Breakout의 클래식 메커닉이다.

## 조작

| 키 | 동작 |
|----|------|
| → | 패들 오른쪽 이동 |
| ← | 패들 왼쪽 이동 |

- 키를 누르고 있으면 패들이 계속 움직입니다(연속 이동, 한 틱당 7px).
- 패들은 좌/우 끝(x = ±200)을 넘어가지 않습니다.
- 패들의 **오른쪽 끝**에 맞히면 공이 오른쪽 위로 가파르게, **왼쪽 끝**에 맞히면 왼쪽 위로, **중앙**에 맞히면 거의 수직으로 곧게 올라갑니다. 받아치는 위치로 각도를 만드는 것이 손맛의 핵심입니다.

## 룰

- 위쪽 벽돌은 행마다 색과 점수가 다릅니다: 빨강(맨 위) 50점 / 주황 30점 / 노랑 20점 / 초록(맨 아래) 10점.
- 벽돌 32개를 모두 깨면 다음 라운드로 넘어가고, 라운드마다 공 속도가 +1 빨라집니다.
- 공을 받아치지 못해 바닥(y < -175)으로 떨어뜨리면 목숨이 1 줄어듭니다.
- 목숨 3개를 모두 쓰면 "GAME OVER" 배너가 뜨고 게임이 끝납니다.
- 패들에 맞을 때마다 공이 조금씩 빨라져(공속도 +0.3) 랠리가 길어질수록 받아치기 어려워집니다.

## 게임 흐름

초록 깃발 클릭 → 점수 0 / 목숨 3 / 라운드 1로 초기화, 8×4 벽돌 격자 생성, 공이 패들 위에서 잠깐 멈췄다 위로 출발 → 화살표로 패들을 움직여 공을 받아치며 벽돌을 깸 → 벽돌을 모두 깨면 라운드 +1·공 속도 ↑로 새 격자가 깔리고 공이 다시 발사 → 공을 떨어뜨리면 목숨 -1 후 패들 위에서 재발사 → 목숨 0이 되면 "GAME OVER" 배너가 뜨고 정지 → 깃발 다시 클릭으로 즉시 재시작.

## 핵심 메커닉 (각도 컨트롤 + 3면 반사)

- **각도 컨트롤**: 패들 반사 시 `오프셋 = (공의 x − 패들의 x) / 패들반폭(35)`으로 충돌 위치를 −1..+1로 정규화하고, `공dx = 오프셋 × 공속도`로 가로 속도를, `공dy = 공속도`(항상 양수=위로)로 세로 속도를 정합니다. 패들 폭이 70(반폭 35)이라 패들 양끝에서 오프셋이 정확히 ±1이 됩니다.
- **3면 벽 반사**: 좌/우 벽(x = ±232)에서는 `공dx × −1`로 X를 뒤집고, 위 벽(y = 170)에서는 `공dy = −1 × abs(공dy)`로 항상 아래로 보냅니다. 반사 직후 공을 벽 안쪽으로 살짝 밀어 끼임을 방지합니다.
- **벽돌 격자 (클론)**: `벽돌생성` 방송을 받으면 벽돌 스프라이트가 8열 × 4행 = 32개의 클론을 좌표·행을 복사하며 생성하고 `벽돌수`를 32로 맞춥니다. 각 클론은 행(0~3)에 따라 코스튬(빨강/주황/노랑/초록)을 고르고, 공에 닿으면 행별 점수를 더하고 `벽돌수`를 −1 한 뒤 자신을 삭제합니다. `벽돌수`가 0이 되면 `다음라운드`를 방송합니다.
- **목숨·라운드**: 공이 바닥(y < -175)으로 떨어지면 `목숨 −1`. 목숨이 1 미만이면 `결과=1`·`게임상태=0`으로 게임오버, 그렇지 않으면 `공발사`로 재발사. 전 벽돌을 깨면 `다음라운드`가 `라운드 +1`·`공속도 +1` 후 격자를 다시 깔고 공을 재발사합니다.
