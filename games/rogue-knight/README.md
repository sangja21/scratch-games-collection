# 로그 나이트

> 기사 한 명으로 한 화면짜리 방에 갇혀, 다가오는 적을 검으로 베어 전멸시킨다. 적은 **약한 적·중간 적·강한 적 3종류** — 방이 깊어질수록(1번 방은 약한 적만, 2번 방은 약+중, 3번 방부터는 중+강) 더 크고 단단하고 빠른 적이 섞여 나온다. 적을 때릴 때마다 맞은 자리에 **데미지 숫자가 통! 하고 떠오른다.** 방을 비우면 강화 셋 중 하나를 골라 더 세진 채 다음 방으로! 체력이 0이 되면 1번 방부터 다시 시작하는 로그라이트 액션 게임이다.

이 게임의 진짜 재미는 **"코드의 숫자를 직접 바꿔보며 내 게임으로 만드는 것"** 이다. 공격력·점프력·적 속도 같은 모든 값이 한글 변수로 정리되어 있어서, 숫자 하나만 바꿔도 게임이 확 달라진다. 아래 [개조 손잡이](#️-개조-손잡이-바꿔보세요)와 [개조 미션 사다리](#-개조-미션-사다리)를 따라 하나씩 도전해 보자.

---

## 조작법

| 키 | 동작 |
|----|------|
| ← → | 좌우 이동 |
| ↑ 또는 Space | 점프 (공중에서 한 번 더 누르면 **2단 점프**!) |
| Z | 검 휘두르기 (기사 앞쪽 적을 벤다) |
| X | 대시 (바라보는 방향으로 순간 이동, 회피용) |
| 1 / 2 / 3 | 방 클리어 후 강화 고르기 |

---

## 게임 흐름

1. **시작** — 초록 깃발을 누르면 1번 방이 만들어지고, 적이 방목표 수만큼 나타난다.
2. **전투** — 좌우로 움직이고 점프·대시로 피하면서, Z로 검을 휘둘러 적을 벤다. 적은 색과 크기로 종류를 구분할 수 있다(작고 옅은 약한 적 / 보통 중간 적 / 크고 진한 강한 적). 강한 적일수록 체력이 많아 여러 번 베야 하고 더 빠르게 쫓아온다. 검에 맞은 적의 자리에 **데미지 숫자(공격력)가 떠올랐다 사라진다** — 얼마나 아프게 때렸는지 한눈에 보인다. 적에 닿으면 체력이 1 깎이고 잠깐 무적이 된다.
3. **클리어 → 강화** — 방의 적을 다 잡으면 화면 가운데에 강화 카드가 뜬다. `1 공격력+`, `2 체력+`, `3 이동속도+` 중 하나를 키로 고른다.
4. **더 깊이** — 다음 방으로 넘어가면 적이 한 마리 더 많아지고(방목표 = 2 + 방번호), 더 센 종류가 섞여 나온다. 돌수록 어려워진다.
5. **죽으면 처음부터** — 체력이 0이 되면 GAME OVER. 초록 깃발을 다시 누르면 능력치가 기본값으로 돌아가 1번 방부터 다시 시작한다.

## ⚙️ 개조 손잡이 (바꿔보세요!)

게임을 바꾸는 모든 숫자는 **무대(Stage)의 초록 깃발 스크립트 한 곳**에 모여 있다. 코드 어디에도 숨겨진 숫자가 없으니, 여기 18개 손잡이만 돌리면 게임이 달라진다.

> 찾는 법: Scratch 에서 **무대**를 클릭 → 코드 탭 → 맨 위 `초록 깃발을 클릭했을 때` 묶음을 보면 `공격력 을(를) 1 로 정하기` 같은 블록들이 쭉 있다. 그 숫자를 바꾸면 끝!

| 손잡이 (변수) | 기본값 | 무슨 뜻? / 바꾸면 일어나는 일 |
|---------------|:------:|-------------------------------|
| **공격력** | 1 | 검 한 번에 적 체력을 깎는 양. 올리면 적을 더 빨리 처치한다. 강화 "공격력+" 가 이 값을 올린다 |
| **최대체력** | 5 | 기사 시작 체력. 올리면 더 오래 버틴다. 강화 "체력+" 가 이 값을 올리고 체력도 채워준다 |
| **체력** | 5 | 지금 남은 체력(최대체력으로 시작). 적에 닿으면 1 줄고, 0이 되면 게임오버 |
| **이동속도** | 4 | 좌우로 한 번에 움직이는 거리. 올리면 빨라진다. 강화 "이동속도+" 가 올린다 |
| **점프력** | 12 | 점프할 때의 힘. 올리면 더 높이 뛴다 |
| **중력** | -1 | 매 순간 아래로 당기는 힘. 절댓값을 키우면(예: -2) 더 빨리, 묵직하게 떨어진다 |
| **점프횟수상한** | 2 | 착지 후 가능한 점프 수. 2면 2단 점프, 1이면 보통 점프, 3이면 3단 점프! |
| **대시거리** | 90 | 대시 한 번에 순간 이동하는 거리. 올리면 더 멀리 피한다 |
| **대시쿨** | 30 | 대시하고 다시 쓸 때까지 기다리는 시간(약 1.2초). 줄이면 자주 대시한다 |
| **무적시간** | 25 | 맞은 뒤 무적이 되는 시간(약 1초). 연속으로 맞아 즉사하는 걸 막는다. 올리면 더 안전 |
| **약한적_체력** | 1 | 약한 적(작고 옅은 색) 한 마리의 체력. 올리면 약한 적도 여러 번 베야 한다 |
| **약한적_속도** | 2.0 | 약한 적이 다가오는 빠르기. 잽싸게 달려든다. 올리면 더 빨라지고, 줄이면 거북이가 된다 |
| **중간적_체력** | 2 | 중간 적(보통 크기·색) 한 마리의 체력 |
| **중간적_속도** | 1.3 | 중간 적이 다가오는 빠르기 |
| **강한적_체력** | 4 | 강한 적(크고 진한 색) 한 마리의 체력. 가장 단단하다 |
| **강한적_속도** | 0.8 | 강한 적이 다가오는 빠르기. 느릿느릿 묵직하게 다가온다 |
| **강화량** | 1 | 강화를 고를 때 능력치가 오르는 양. 올리면 강화 한 번이 화끈해진다 |
| **방목표** | 3 | 지금 방에서 잡아야 할 적 수. 방을 깰수록 자동으로 늘어난다 |

## 🎯 개조 미션 사다리

쉬운 것부터 어려운 것까지. **하나씩 깨보면서 "코드를 망가뜨리고 고치는" 재미**를 느껴보자. 망가져도 괜찮다 — 초록 깃발만 다시 누르면 된다!

### 0층 — 숫자 하나 바꾸기 (무대 초록 깃발 스크립트)
- **세진 기사**: `공격력` 을 `1` → `3` 으로. 적이 훨씬 빨리 죽는다. **적을 때려서 머리 위에 뜨는 데미지 숫자가 `1` 에서 `3` 으로 바뀌는지** 직접 확인해 보자 — 데미지 팝업은 `공격력` 을 그대로 보여준다!
- **두부 멘탈**: `최대체력` 을 `5` → `1` 로. 한 대만 맞아도 위험! (난이도 급상승)
- **슈퍼 점프**: `점프력` 을 `12` → `20` 으로. 천장까지 날아오른다.
- **느릿느릿 약골**: `약한적_속도` 를 `2` → `0.3` 으로. 잽싸던 약한 적이 거북이가 된다.
- **강한적을 두부로**: `강한적_체력` 을 `4` → `1` 로. 가장 센 적도 한 방에 가깝게 쓰러진다.
- **약한적이 더 빠르게**: `약한적_속도` 를 `2` → `5` 로. 작은 적들이 떼로 무섭게 몰려와 정신이 없다!
- **3단 점프**: `점프횟수상한` 을 `2` → `3` 으로. 공중에서 세 번 뛴다!

### 1층 — 값을 식으로 묶기 (관계 만들기)
- **잡을수록 세지는 검**: `적` 스프라이트의 `내체력 을 (0 - 공격력) 만큼 바꾸기` 블록을, `내체력 을 (0 - (1 + (총처치 / 3))) 만큼 바꾸기` 로 바꾼다. 많이 잡을수록 검이 강해진다.
- **달밤 체조**: 무대의 `방빌드` 스크립트에서 `이동속도 = 4 + 방번호` 가 되도록 손봐서, 깊은 방일수록 빨라지게.
- **묵직 점프**: 무대 초록 깃발에서 `중력` 을 `(0 - 점프력) / 12` 로 묶어, 점프력을 키우면 중력도 같이 세지게.

### 2층 — if 조건 더하기
- **마지막 발악**: `기사` 의 피격 감시 스크립트에 `만약 체력 = 1 이면 → 무적 을 60 으로 정하기` 를 추가. 체력 1 남으면 잠깐 무적이 된다.
- **광전사 모드**: `기사` 이동 스크립트에 `만약 체력 = 1 이면 → 이동속도 = 8` 을 추가해, 위험할 때 빨라지게.
- **보스 방**: 무대 `방빌드` 에 `만약 (방번호 나누기 3의 나머지) = 0 이면 → 강한적_체력 = 강한적_체력 × 2` 를 넣어, 3번째 방마다 강한 적을 더 단단하게.

### 3층 — 규칙 자체를 바꾸기 (분기)
- **회복 강화 추가**: `강화패널` 스크립트에 `만약 4키 누름 → 체력 을 최대체력 만큼 바꾸기` (풀회복 선택지)를 더하고, 카드 그림에 4번 칸을 그려 넣는다.
- **랜덤 강화**: 강화 선택을 키 입력 대신 `1부터 3 사이의 난수` 로 자동 결정하게 바꿔, 운에 맡겨보자.

### 4층 — 새 메커닉 만들기 (구조)
- **점프 적 추가**: `적` 클론에 VY·중력을 달아(기사 점프 로직을 참고) 발판을 뛰어다니는 점프형 적으로 개조.
- **새 방 추가**: `발판` 스프라이트의 `방번호 나누기 3의 나머지` 를 `나누기 4의 나머지` 로 늘리고, 4번째 발판 좌표 세트를 추가한다.
- **화려한 데미지 숫자**: `데미지` 스프라이트의 클론 스크립트에서 떠오르는 속도(`y 좌표를 4만큼 바꾸기`)나 사라지는 빠르기(`반투명 효과를 8만큼 바꾸기`)를 바꿔, 데미지 팝업이 더 높이·천천히 떠오르게 연출해 보자. `데미지표시값` 에 `× 2` 를 곱해 "표시만 두 배"인 가짜 폭딜 숫자를 띄워볼 수도 있다.
