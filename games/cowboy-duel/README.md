# 카우보이 결투 (Cowboy Duel)

서부 석양 아래 두 카우보이가 마주 선다. "준비..." 후 언제 뜰지 모르는 **"DRAW!"** 신호가 번쩍이면 즉시 스페이스를 눌러 사격한다. AI보다 빠르면 라운드 승리, 신호 전에 누르면 부정출발(반칙)로 그 라운드 패배. AI는 라운드가 갈수록 빨라진다. 먼저 3라운드를 이기면 최종 승리. 순수 반응속도 대결이다.

## 조작

| 키 | 동작 |
|----|------|
| 스페이스 | 사격 |

- 신호("DRAW!")가 번쩍인 **뒤에** 누르세요. 신호 전에 누르면 부정출발(반칙)로 그 라운드를 집니다.
- 한 라운드는 스페이스 한 번이면 끝납니다. 빠를수록 좋습니다.

## 룰

- "DRAW!" 신호 후 내가 AI보다 빨리 스페이스를 누르면 라운드 승리, 느리거나 못 누르면 패배.
- 신호가 뜨기 전에 스페이스를 누르면 부정출발(FOUL) → 그 라운드 즉시 패배(AI 승점).
- 5판 3선승. 먼저 3라운드를 이기면 VICTORY, AI가 먼저 3라운드를 이기면 DEFEAT.
- AI 반응시간은 라운드마다 무작위이고 라운드가 진행될수록 빨라집니다(1라운드 0.45~0.70초 → 5라운드 0.28~0.50초). 다만 0.28초 미만으로는 빨라지지 않아, 집중하면 이길 수 있는 난이도입니다.

## 게임 흐름

초록 깃발 클릭 → 점수 0:0, 1라운드 시작 → 화면에 "준비..." 표시 → 1.5~4초의 무작위 대기(언제 신호가 뜰지 모름) → "DRAW!" 번쩍 + 총소리 → 먼저 사격한 쪽이 라운드 승리(승자는 사격 자세, 패자는 쓰러짐) → WIN/LOSE/FOUL 배너 → 점수 갱신 → 다음 라운드 → 한쪽이 3승에 도달하면 VICTORY/DEFEAT 배너가 뜨고 매치 종료 → 깃발 다시 클릭으로 즉시 재시작.

화면 상단에는 내승 / AI승 / 베스트ms(매치 중 최고 반응 기록) / 내반응ms(이번 라운드 기록) / 라운드 번호가 표시됩니다.

## 핵심 메커닉 (반응시간 측정 + 부정출발 판정)

- **무작위 대기**: 매 라운드 `대기시간 = 무작위(1.5~4.0초)`를 변수에 한 번 저장합니다. 무작위 값은 평가할 때마다 달라지므로 반드시 변수에 담아 대기 루프와 부정출발 판정에서 같은 값을 씁니다.
- **반응시간(ms) 측정**: 대기 시작 시 `timer`를 리셋하고, 신호가 뜨는 순간 `신호시각 ← timer`로 기준점을 잡습니다. 스페이스를 누른 순간의 반응시간은 `round((timer − 신호시각) × 1000)` ms로 계산합니다. 같은 timer 기준을 끝까지 사용해 시점이 일관됩니다.
- **부정출발 판정**: `반복(까지) ((timer ≥ 대기시간) 또는 (스페이스))` 루프를 돌린 뒤, 빠져나온 직후 `만약 (스페이스) 그리고 (timer < 대기시간)`이면 신호 전에 누른 것 → 부정출발(반칙 패). 스페이스를 미리 누르고 있으면 timer가 0에 가까운 시점에 루프가 즉시 빠져나오고 곧바로 반칙 처리되므로, **키를 계속 누르고 있어 자동으로 이기는 치트는 통하지 않습니다**(눌러둔 채로는 매 라운드 반칙).
- **선착 판정**: 신호 후 `…까지 기다리기 ((스페이스) 또는 (timer ≥ 신호시각 + AI반응))`. 빠져나온 뒤 `만약 (스페이스) 그리고 ((timer − 신호시각) < AI반응)`이면 내가 더 빨랐다 → 승리, 아니면 AI 승리. 동시(마감 시각)는 AI 승으로 처리됩니다.
- **AI 난이도 스케일링**: `AI반응 = 무작위(0.45~0.70) − (라운드 − 1) × 0.05`, 하한 0.28초. 라운드가 갈수록 AI가 빨라지지만 0.28초 밑으로는 내려가지 않습니다.
- **베스트 기록**: 정상 승리(라운드결과=1)일 때만, 그리고 이번 기록이 기존 베스트보다 작을 때만 베스트ms를 갱신합니다. 패배·반칙 기록은 베스트에서 제외됩니다.

## 실행

1. `카우보이결투.sb3` 다운로드
2. [scratch.mit.edu](https://scratch.mit.edu) → 만들기 → 파일 → 컴퓨터에서 가져오기
3. 초록 깃발 ▶ 클릭

```bash
cd games/cowboy-duel
python3 build.py
# → 카우보이결투.sb3 갱신
```
