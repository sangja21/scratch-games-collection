# 스네이크 (Snake)

방향키로 16px 격자 위를 움직이는 뱀을 조종해 사과를 먹는다. 사과를 먹을 때마다 꼬리가 한 칸씩 길어지고 점수 +1. 벽이나 자기 꼬리에 부딪히면 게임오버. 1976 아케이드 Blockade / Nokia "Snake" 의 클래식 메커닉이다.

## 조작

| 키 | 동작 |
|----|------|
| → | 오른쪽으로 진행 |
| ↑ | 위로 진행 |
| ← | 왼쪽으로 진행 |
| ↓ | 아래로 진행 |

- 뱀은 매 스텝(0.18초)마다 진행 방향으로 한 칸(16px)씩 자동으로 움직입니다. 방향키는 다음에 꺾을 방향을 예약합니다.
- 진행 중인 방향의 정반대로는 꺾을 수 없습니다(오른쪽으로 가는 중엔 왼쪽 입력 무시). 자기 목으로 곧장 박는 사고를 막아줍니다.

## 룰

- 사과를 먹으면 점수 +1, 꼬리가 한 칸 길어집니다.
- 벽(보드 바깥)에 닿거나 자기 꼬리에 닿으면 게임오버.
- 점수가 5점을 넘으면 점수가 오를수록 뱀이 점점 빨라집니다(스텝 간격 0.18초 → 최소 0.07초).
- 최고기록은 게임오버 사이에도 세션 동안 유지됩니다.

## 게임 흐름

초록 깃발 클릭 → 길이 3칸 뱀이 보드 좌중앙에서 시작 → 방향키로 사과를 먹으며 꼬리를 늘림 → 벽 또는 꼬리 충돌 시 "GAME OVER" 배너 → 깃발 다시 클릭으로 즉시 재시작(변수·리스트·클론 초기화).

## 핵심 메커닉 (꼬리 추종)

머리는 매 스텝 지나온 좌표를 `궤적X`/`궤적Y` 리스트 맨 앞에 기록합니다. 꼬리 세그먼트는 클론이며, 각 클론은 자기 순번(`세그번호`)만큼 지연된 과거 좌표(`item(세그번호+1)`)를 읽어 머리가 지나온 길을 그대로 따라갑니다. 머리가 궤적 기록을 마친 뒤 `꼬리갱신`을 방송하고 꼬리가 그때 위치를 읽으므로, 꼬리는 항상 머리 바로 뒤에 끊김 없이 이어집니다. 리스트는 `길이+5` 항목으로 잘라 무한 증가를 막습니다.

## 실행

1. `스네이크.sb3` 다운로드
2. [scratch.mit.edu](https://scratch.mit.edu) → 만들기 → 파일 → 컴퓨터에서 가져오기
3. 초록 깃발 ▶ 클릭

## 다시 빌드

```bash
cd games/snake
python3 build.py
# → 스네이크.sb3 갱신
```

- 자산: 인라인 SVG 5종 (보드·머리·꼬리·apple·게임오버) + pop.wav
- 리스트를 실제로 사용하는 첫 게임 (`궤적X`/`궤적Y` — insert/trim/item-of 기반 꼬리 관리)
