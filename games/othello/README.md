# 오델로 (리버시) · Othello

2인 플레이(흑 ● vs 백 ○) 보드게임. 빈 칸을 클릭해 돌을 놓으면 8방향으로 끼인 상대 돌이 모두 뒤집힌다. 합법 수(뒤집을 수 있는 자리)에만 둘 수 있고, 둘 곳이 없으면 자동 패스, 양쪽 다 없으면 게임 종료 후 돌 수로 승패를 가린다.

## 조작
- **빈 칸 클릭** → 돌 놓기 (흑 선공)
- 합법 수가 아니면 무시됨
- 점수(흑/백)와 차례·결과는 화면 상단 안내에 표시

## 구현 메모
- **보드**: 길이 64 리스트 `보드` (0=빈칸, 1=흑, 2=백). 칸번호 `n = 행*8 + 열 + 1`.
- **렌더**: `돌` 스프라이트 클론 64개. 각 클론이 칸번호로 위치를 잡고 `보드[칸번호]` 값에 따라 흑/백 코스튬·표시.
- **입력/로직 분리**: 클릭은 마우스 좌표 → (행,열)만 계산해 `착수` 방송. 실제 두기 로직은 `착수` 수신부에. (headless 검증을 위해 의도적으로 분리)
- **핵심 루틴**(broadcast-and-wait 동기 호출):
  - `수계산` — (행,열,차례,상대)에 대해 8방향 광선 스캔으로 뒤집을 칸 목록과 합법 여부 계산. 픽셀 `touching` 대신 **좌표 비교**(검증 가능).
  - `있는수확인` — 현재 차례가 둘 수 있는 빈 칸이 하나라도 있는지.
  - `차례전환` — 점수 재집계 → 차례 교대(패스/종료 처리) → 안내문 갱신.

## 검증
headless scratch-vm 으로 JS 규칙 레퍼런스(오라클)와 대조 완료:
- 초기 배치, 8방향 뒤집기, 점수, 차례 교대·패스 — 60수 풀게임 재생 **불일치 0건**
- 게임 종료(게임상태=0) 및 승자 판정 일치
- 돌 클론 64개의 위치·색·표시가 보드 상태와 일치 (위치/표시 오류 0)

> 참고: 이 프로젝트의 주 방향(초등학생 대상 직관적 액션)과는 다른 보드·전략 장르의 실험적 게임이다.
