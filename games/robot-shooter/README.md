# 로봇 슈터

> **탑뷰 아레나 · 자동 미사일 · 레이저 · 투척 폭탄 · 스킬 아이콘 UI.**  
> 조준·미사일은 자동. 플레이어는 **이동 + 스킬(Z/X)**.  
> 변수 모니터 없음 — **체력 하트 · 점수판 · 스킬 아이콘 쿨** 로 표시.

---

## 조작

| 입력 | 동작 |
|------|------|
| ← → ↑ ↓ | 이동 |
| (자동) | 미사일 → 가장 가까운 적 |
| (자동) | 레이저 (강화 `레이저+` 후) |
| **Z** | **레이저 스킬(메가빔)** — 고관통 강타 |
| **X** | **폭탄 투척** — 날아가다 **펑!** 광역 폭발 |
| 1 / 2 / 3 | 웨이브 클리어 후 랜덤 강화 |

---

## 보스전 (웨이브 5, 10, 15, …)

- **5의 배수 웨이브**마다 잡몹 대신 **보스 1기**
- 하단 **보스 체력 게이지** 표시
- 티어 `floor(웨이브/5)` 회차마다 무기 강화·다양화:

| 회차 | 웨이브 | 패턴 |
|------|--------|------|
| 1 | 5 | 부채 · 더블부채 · 링12 |
| 2 | 10 | + 산탄 · 스파이럴 · 링16 |
| 3 | 15 | + 십자8방 · 링20 |
| 4 | 20 | + 플라워(링+부채) |
| 5+ | 25+ | 전부 강화 · 링24 · 저체력 광폭 |

- 탄수·연사·탄속·탄 크기·보스 이동·체력 모두 티어 비례
- 저체력 시 쿨 단축 + 강한 패턴 강제
- 체력: `기본 + 웨이브×8 + 티어×12`
- 처치 시 점수 +15 → 강화 선택 → 다음 웨이브

## 비주얼 / 사운드

- 캐릭터·적·보스·탄환·배경: **메카 슈터 톤 PNG 에셋** (`assets/gen/`)
- **주인공 보행 애니**: idle + walk 4프레임 × 좌/우 (`walk_r0~4`, `walk_l0~4`). 이동 중 사이클, 정지 시 idle.
- 효과음: Python 합성 **대포·포격 톤** (균열 crack + 저음 붐 + 잔향 럼블). 미사일/보스 함포/폭탄·격파 모두 중저음 포성 계열
- **BGM**: `assets/bgm.mp3` (Suno `robot_shooter_02`, Stage에서 루프 재생). 음량 변수 `브금볼륨`(기본 55%)

> 보행 애니 실험이 마음에 안 들면  
> `games/robot-shooter/.rollback/player-walk-attempt/ROLLBACK.md` 참고해 롤백 가능.

## UI (모니터 없음)

| UI | 위치 | 내용 |
|----|------|------|
| 체력 하트 | 좌상 | 빨간 원 0~5개 |
| 점수판 | 상단 | `W웨이브  S점수` 말풍선 |
| **Z 레이저 아이콘** | 하단 | 쿨 중 반투명(고스트) |
| **X 폭탄 아이콘** | 하단 | 쿨 중 반투명(고스트) |

아이콘이 **선명하면 사용 가능**, 흐리면 쿨타임 중.

---

## 폭탄

1. X → 가장 가까운 적 방향으로 **폭탄 투척**  
2. 날아감  
3. **펑!** 폭발 이펙트 + 반경 안 적에게 공격력×3  

---

## 무기 / 스킬

| 종류 | 설명 |
|------|------|
| 미사일 | 시작부터 자동 |
| 레이저 (자동) | `레이저+` 강화 시 **동시 발사량↑** (레벨=발 수) + 연사 소폭↑ |
| Z 메가빔 | 쿨 ~8초, 공격력×4, 관통 |
| X 폭탄 | 쿨 ~10초, 투척 후 광역 |
