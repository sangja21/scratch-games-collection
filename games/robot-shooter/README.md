# 로봇 슈터

> **탑뷰 아레나 · 자동 미사일 · 레이저 · 투척 폭탄 · 스킬 아이콘 UI.**  
> 조준·미사일은 자동. 플레이어는 **이동 + 스킬(Z/X)**.  
> 변수 모니터 없음 — **체력 하트 · 점수판 · 스킬 아이콘 쿨** 로 표시.

---

## 조작

| 입력 | 동작 |
|------|------|
| ← → ↑ ↓ | 이동 |
| (자동) | 미사일 → 가장 가까운 적 |
| (자동) | 레이저 (강화 `레이저+` 후) |
| **Z** | **레이저 스킬(메가빔)** — 고관통 강타 |
| **X** | **폭탄 투척** — 날아가다 **펑!** 광역 폭발 |
| 1 / 2 / 3 | 웨이브 클리어 후 랜덤 강화 |

---

## 보스전 (웨이브 5, 10, 15, …)

- **5의 배수 웨이브**마다 잡몹 대신 **보스 1기**
- 상단 좌우 순찰 + **부채꼴 다연장** 사격 (웨이브 오를수록 탄 수↑·간격↓·탄속↑)
- **필살기**: 웨이브 10+ 부터 주기적으로 **12방향 전방위 탄막** (저체력이면 더 자주)
- 체력: `보스기본체력 + 웨이브×8`
- 처치 시 점수 +15 → 강화 선택 → 다음 웨이브

## 비주얼 / 사운드

- 캐릭터·적·보스·탄환·배경: **메카 슈터 톤 PNG 에셋** (`assets/gen/`)
- **주인공 보행 애니**: idle + walk 4프레임 × 좌/우 (`walk_r0~4`, `walk_l0~4`). 이동 중 사이클, 정지 시 idle.
- 효과음: 장난감 뿅 제거, **중저음 산업·군사 톤** (미사일 추진, 빔 포, 함포, 장약 폭발)

> 보행 애니 실험이 마음에 안 들면  
> `games/robot-shooter/.rollback/player-walk-attempt/ROLLBACK.md` 참고해 롤백 가능.

## UI (모니터 없음)

| UI | 위치 | 내용 |
|----|------|------|
| 체력 하트 | 좌상 | 빨간 원 0~5개 |
| 점수판 | 상단 | `W웨이브  S점수` 말풍선 |
| **Z 레이저 아이콘** | 하단 | 쿨 중 반투명(고스트) |
| **X 폭탄 아이콘** | 하단 | 쿨 중 반투명(고스트) |

아이콘이 **선명하면 사용 가능**, 흐리면 쿨타임 중.

---

## 폭탄

1. X → 가장 가까운 적 방향으로 **폭탄 투척**  
2. 날아감  
3. **펑!** 폭발 이펙트 + 반경 안 적에게 공격력×3  

---

## 무기 / 스킬

| 종류 | 설명 |
|------|------|
| 미사일 | 시작부터 자동 |
| 레이저 (자동) | `레이저+` 강화 후 보조 연사 |
| Z 메가빔 | 쿨 ~8초, 공격력×4, 관통 |
| X 폭탄 | 쿨 ~10초, 투척 후 광역 |

---

공격력+ · 연사+ · 여러발+ · 관통+ · 레이저+ · 스킬쿨-
