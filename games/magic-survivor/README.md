# 마법 생존

> 마법사 한 명으로 한 화면짜리 아레나에 갇혀, 사방에서 **끝없이 몰려오는 적 무리를 자동 마법으로 쓸어담는** 뱀파이어-서바이벌 라이트. 조준은 필요 없다 — 마법탄은 일정 간격마다 **가장 가까운 적에게 알아서 날아가니**, 너는 화살표키로 8방향을 누비며 **피하는 데만** 집중하면 된다. 적은 약·중·강 3종(슬라임/박쥐/골렘)이고, **오래 버틸수록**(생존 시간이 길수록) 더 세고 빠른 적이 섞여 나온다. 적을 잡으면 경험치 보석이 떨어지고, 가까이 가면 빨려온다. 경험치 바가 차면 **레벨업 → 강화 셋 중 하나**를 골라 점점 강해진다. 체력이 0이 되면 GAME OVER. **점수는 곧 생존 시간(초)** 이다.

이 게임의 진짜 재미는 **"코드의 숫자를 직접 바꿔보며 내 게임으로 만드는 것"** 이다. 공격력·발사 속도·적 체력 같은 모든 값이 한글 변수로 정리되어 있어서, 숫자 하나만 바꿔도 게임이 확 달라진다. 아래 [개조 손잡이](#️-개조-손잡이-바꿔보세요)와 [개조 미션 사다리](#-개조-미션-사다리)를 따라 하나씩 도전해 보자.

## 조작법

| 키 | 동작 |
|----|------|
| ← → ↑ ↓ | 8방향 이동 (피하기만! 조준은 자동) |
| (자동) | 마법탄이 `발사간격`마다 가장 가까운 적에게 알아서 날아간다 |
| 1 / 2 / 3 / 4 | 레벨업 후 강화 고르기 (`1` 마법공격력+ · `2` 발사간격- · `3` 이동속도+ · `4` 여러발+) |

## 게임 흐름

1. **시작** — 초록 깃발을 누르면 능력치가 기본값으로 맞춰지고, 마법사가 화면 중앙(0,0)에 선다. 곧 가장자리에서 적이 쏟아지기 시작한다.
2. **전투** — 화살표키로 움직여 적을 피한다. 마법탄은 `발사간격`마다 자동으로 가장 가까운 적에게 날아간다. 적은 색과 크기로 종류를 구분한다(작고 초록인 **슬라임** / 보라 **박쥐** / 크고 회갈색인 **골렘**). 센 적일수록 체력이 많아 여러 번 맞혀야 하고, 맞은 적의 자리에 **데미지 숫자(마법공격력)가 통! 하고 떠오른다.** 적에 닿으면 체력이 1 깎이고 잠깐 무적이 된다.
3. **보석 → 레벨업** — 적을 잡으면 청록 경험치 보석이 떨어진다. `흡수범위` 안에 들어가면 자석처럼 빨려와 경험치가 쌓이고, 경험치 바가 차면 화면 가운데에 강화 카드가 뜬다. `1 마법공격력+`, `2 발사간격-`, `3 이동속도+`, `4 여러발+`(마법탄을 한 번에 부채꼴로 더 많이 발사) 중 하나를 키로 고르면 전투가 재개된다. `4 여러발+`를 여러 번 고르면 한 번에 쏘는 마법탄이 점점 늘어 화면을 뒤덮는다.
4. **점점 어려워짐** — 생존 시간이 `난이도주기`(15초)를 넘길 때마다 난이도 `단계`가 1 올라, 등장하는 적이 **약만 → 약·중 → 중·강** 으로 점점 거칠어진다. 게다가 같은 종류의 적도 단계가 깊어질수록 **체력·속도가 같이 올라가고**(`적체력증가`·`적속도증가`) 더 **자주 쏟아진다**(`스폰감소`). 그래서 마법사가 강화로 세지는 만큼 적도 세져서, 공격력만 올린다고 학살할 수 없다. 오래 버틸수록 화면이 적으로 가득 찬다.
5. **죽으면 처음부터** — 체력이 0이 되면 GAME OVER 배너가 뜨고 생존 시간(점수)이 보인다. 초록 깃발을 다시 누르면 능력치가 기본값으로 돌아가 처음부터 다시 시작한다.

## ⚙️ 개조 손잡이 (바꿔보세요!)

게임을 바꾸는 모든 숫자는 **무대(Stage)의 초록 깃발 스크립트 한 곳**에 모여 있다. 코드 어디에도 숨겨진 숫자가 없으니, 여기 24개 손잡이만 돌리면 게임이 달라진다.

> 찾는 법: Scratch 에서 **무대**를 클릭 → 코드 탭 → 맨 위 `초록 깃발을 클릭했을 때` 묶음을 보면 `마법공격력 을(를) 1 로 정하기` 같은 블록들이 쭉 있다. 그 숫자를 바꾸면 끝!

### 난이도 스케일링 손잡이 (마법사가 세지는 만큼 적도 세진다)

> 마법사는 강화로 점점 강해지므로, 적도 **단계가 깊어질수록** 같이 강해져야 학살을 막을 수 있다. 이 네 손잡이가 그 균형을 잡는다. (적 한 마리의 실제 체력 = `종류 기본체력 + 단계 × 적체력증가`, 속도도 같은 방식)

| 손잡이 (변수) | 기본값 | 무슨 뜻? / 바꾸면 일어나는 일 |
|---------------|:------:|-------------------------------|
| **적체력증가** | 1 | 난이도 `단계`가 1 오를 때마다 **모든 적**에게 더해지는 체력. 올리면 후반 적이 급격히 단단해진다(공격력만 올려선 학살 불가). `0` 으로 두면 적 체력이 안 자란다 |
| **적속도증가** | 0.05 | 단계당 모든 적에게 더해지는 속도. 올리면 후반에 적이 점점 빨라져 피하기 어려워진다 |
| **스폰감소** | 0.06 | 단계당 줄어드는 스폰 간격(초). 올리면 단계가 오를수록 적이 훨씬 더 자주 쏟아진다 |
| **스폰간격최소** | 0.4 | 스폰 간격이 아무리 줄어도 이 값 밑으로는 안 내려간다(적 폭주 안전장치). 낮추면 후반 물량이 극단적으로 많아진다 |

## 🎯 개조 미션 사다리

쉬운 것부터 어려운 것까지. **하나씩 깨보면서 "코드를 망가뜨리고 고치는" 재미**를 느껴보자. 망가져도 괜찮다 — 초록 깃발만 다시 누르면 된다!

### 0층 — 숫자 하나 바꾸기 (무대 초록 깃발 스크립트)
- **속사포 마법사**: `발사간격` 을 `0.6` → `0.2` 로. 마법탄이 따발총처럼 쏟아진다.
- **관통 마법**: `관통` 을 `1` → `5` 로. 한 발이 일렬의 적 다섯을 뚫는다.
- **삼연발**: `추가발사` 를 `1` → `3` 으로. 부채꼴 3발이 동시에 날아간다.
- **세진 마법**: `마법공격력` 을 `1` → `3` 으로. **적을 때려서 뜨는 데미지 숫자가 `1` 에서 `3` 으로 바뀌는지** 직접 확인해 보자 — 데미지 팝업은 `마법공격력` 을 그대로 보여준다!
- **두부 멘탈**: `최대체력` 을 `5` → `1` 로. 한 대만 맞아도 위험! (난이도 급상승)
- **거북이 골렘**: `강한적_속도` 를 `0.6` → `0.2` 로. 골렘이 더 느릿느릿 다가온다.
- **자석 장갑**: `흡수범위` 를 `90` → `200` 으로. 보석이 화면 멀리서도 빨려온다.
- **지옥문**: `스폰간격` 을 `1.2` → `0.4` 로. 적이 세 배로 쏟아진다(난이도 급상승).

### 1층 — 값을 식으로 묶기 (관계 만들기)
- **잡을수록 빨라지는 손**: 강화카드 `2` 번을 `발사간격 을 (0 - (발사간격 / 10)) 만큼 바꾸기` 로 바꿔, 고를 때마다 **현재 간격의 10%씩** 줄게(곱셈 감소).
- **레벨 비례 공격**: `적` 의 마법탄 피격에서 `내체력 을 (0 - 마법공격력) 만큼 바꾸기` 를 `내체력 을 (0 - (마법공격력 + (레벨 / 2))) 만큼 바꾸기` 로. 레벨이 오를수록 기본 화력이 세진다.
- **스케일링 세게/약하게**: 적이 단계마다 강해지는 정도는 `적체력증가`·`적속도증가` 손잡이로 정해진다(기본 1, 0.05). `적체력증가` 를 `2` 로 올리면 후반이 훨씬 단단해지고, `0` 으로 두면 적 체력이 안 자라 쉬워진다. 내 강화 속도에 맞춰 균형을 맞춰 보자.

### 2층 — if 조건 더하기
- **마지막 발악**: `마법사` 의 피격 감시 스크립트에 `만약 체력 = 1 이면 → 무적 을 60 으로 정하기` 를 추가. 체력 1 남으면 잠깐 무적이 된다.
- **레벨업 폭발**: 무대의 레벨업 처리에 `무적 을 무적시간 으로 정하기` 를 추가해, 레벨업 순간 잠깐 무적으로 숨을 돌린다.
- **후반 가속 폭발**: 적 클론의 단계 스케일링은 `내속도 를 (단계 × 적속도증가) 만큼 바꾸기` 로 들어 있다. 여기에 `만약 단계 >= 5 이면 → 내속도 를 (내속도 + 1) 만큼 바꾸기` 를 더해, 5단계부터 적이 한 번 더 확 빨라지는 "발악 구간" 을 만들어 보자.

### 3층 — 규칙 자체를 바꾸기 (분기)
- **강화 5번 추가(관통+/최대체력+)**: `강화카드` 에 `만약 5키 누름 → 관통 을 1 만큼 바꾸기` (또는 `최대체력 을 강화량 만큼 · 체력 을 강화량 만큼 바꾸기`)를 더하고, 카드 그림에 5번 칸을 그려 넣는다. (이미 `4 여러발+` 까지 들어 있으니 `5` 만 추가하면 된다.)
- **랜덤 강화**: 강화 선택을 키 입력 대신 `1부터 4 사이의 난수` 로 자동 결정하게 바꿔, 운에 맡겨보자.

### 4층 — 새 메커닉 만들기 (구조)
- **유도탄**: `마법탄` 클론의 직진 루프에 매 틱 `적 쪽 보기` 를 한 번 섞어, 살짝 휘어 날아가는 유도 마법으로.
- **회복 보석 추가**: `경험치보석` 을 복제해 `체력보석` 스프라이트를 만들고, 적 처치 시 가끔(`1부터 5 사이의 난수 = 1`) 떨어뜨려 닿으면 `체력 을 1 만큼 바꾸기`.
- **분열 슬라임**: `적` 처치 시 `만약 적종류 = 1 이면 → 작은 슬라임 2마리 스폰` 을 넣어, 약한 적이 죽으며 둘로 쪼개지게(스폰 채널 재사용).
- **화려한 데미지 숫자**: `데미지` 클론의 떠오르는 속도(`y 좌표를 4만큼 바꾸기`)나 사라지는 빠르기(`반투명 효과를 8만큼 바꾸기`)를 바꿔 더 높이·천천히 떠오르게 하거나, `데미지표시값` 에 `× 2` 를 곱해 "표시만 두 배"인 가짜 폭딜 숫자를 띄워보자.

## 학습 포인트 (부가)

수학 학습 콘셉트는 없는 순수 액션 게임이다. 대신 **"코딩 학습"이 핵심 가치**다 — 모든 능력치가 한글 튜닝 변수로 노출되어, 아이가 (0층) 숫자를 바꾸고 → (1층) 값을 식으로 묶고 → (2층) if 조건을 더하고 → (4층) 새 스프라이트를 추가하며, **변수·연산·조건·클론·broadcast and wait** 라는 프로그래밍 개념을 게임을 망가뜨리고 고치며 자연스럽게 체득한다.

특히 이 게임의 심장인 **자동 조준**은 `broadcast 조준요청 and wait` 한 줄로 동작한다 — 모든 적 클론이 "내가 지금까지 중 마법사에게 가장 가까우면 내 위치를 보고한다"를 차례로 실행해, 경쟁 없이 가장 가까운 적을 찾아낸다(최솟값 찾기). 그 밖에 **타이머/누적**(생존시간), **확률적 스폰**(가장자리·종류 난수), **점점 세지는 난이도 곡선**(단계 = 생존시간 ÷ 난이도주기)을 게임 행동으로 만나게 된다.
