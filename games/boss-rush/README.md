# 보스 러시

> 1대 보스 vs 플레이어. 화면 하단 플레이어는 좌/우 화살표로 이동하면서 스페이스로 위쪽으로 직선 사격. 화면 상단의 거대한 보스는 좌↔우로 왕복하면서 체력에 따라 3개 페이즈로 점점 강해지는 탄막을 발사합니다. 플레이어는 보스 탄을 피하며 사격해 보스 체력 100을 깎아 격파해야 해요. 페이즈마다 탄막 패턴이 바뀌니 잘 보고 외워서 피하세요!

## 조작

- **← / →**: 좌우 이동 (벽까지)
- **스페이스**: 위로 총알 발사 (자동 쿨다운)

## 게임 흐름

1. 초록 깃발(▶) 클릭으로 시작.
2. 상단의 거대한 보라색 UFO 보스가 좌↔우로 왕복하면서 탄막을 발사합니다.
3. 보스 체력에 따라 3개 페이즈:
   - **페이즈 1 (체력 100~67%)**: 보스가 천천히 이동하면서 부채꼴 3발을 1초 간격으로 발사.
   - **페이즈 2 (체력 66~34%)**: 보스 속도 ↑, 부채꼴 5발을 0.8초 간격으로 발사.
   - **페이즈 3 (체력 33~0%)**: 보스 매우 빠르게 이동, **원형 16발 전방향 탄막**을 1.2초 간격으로 발사!
4. 플레이어 체력은 5, 보스 탄에 닿으면 −1 (0.5초 무적).
5. 보스 격파 = VICTORY! / 플레이어 체력 0 = GAME OVER.

## 팁

- 페이즈 1에서는 보스 정면 아래만 피하면 됩니다. 보스 양옆에서 사격하면 안전!
- 페이즈 3 원형 탄막은 시작되면 화면 중앙에서 빠져나오기 어려우니, 미리 화면 끝쪽으로 빠져 있다가 탄막 사이 틈으로 회피하세요.
- 플레이어 총알은 보스에 닿으면 −2 데미지. 페이즈 3 진입은 +50발 정도면 충분합니다.
