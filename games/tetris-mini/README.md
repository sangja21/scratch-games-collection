# 테트리스 미니 (Tetris Mini)

7가지 블록(테트로미노)이 8열×14행 보드 위로 떨어진다. 좌/우로 옮기고 위로 돌려서 가로줄을 빈칸 없이 채우면 그 줄이 사라지고 점수가 오른다. 줄을 많이 지울수록 블록이 빨라지고, 블록이 천장까지 쌓이면 게임오버. 1984년 Alexey Pajitnov의 "Tetris"를 초등학생용으로 간소화한 버전이다.

## 조작

| 키 | 동작 |
|----|------|
| ← | 블록을 한 칸 왼쪽으로 |
| → | 블록을 한 칸 오른쪽으로 |
| ↑ | 블록을 시계방향으로 회전 |
| ↓ | 소프트드롭 (누르고 있으면 빨리 내려감) |
| Space | 하드드롭 (바닥까지 한 번에 떨어뜨려 고정) |

←/→/↑ 는 한 번 누름 = 한 칸 이동/한 번 회전입니다(꾹 눌러도 자동 반복 안 됨). 벽·바닥·이미 쌓인 블록에 막히는 이동/회전은 무시됩니다(벽킥 없음).

## 게임 흐름

1. 초록 깃발 ▶ 을 누르면 보드가 비워지고 첫 블록이 맨 위 가운데에서 떨어지기 시작합니다.
2. 블록이 일정 간격(`낙하간격`)마다 한 칸씩 자동으로 내려옵니다. 좌/우/회전으로 자리를 잡고, 더 못 내려가면 그 자리에 굳습니다.
3. 가로줄 8칸이 빈칸 없이 꽉 차면 그 줄이 사라지고 위 줄들이 한 칸씩 내려옵니다. 한 번에 여러 줄도 동시에 지워집니다.
4. 줄을 지울수록 레벨이 오르고 블록 낙하가 빨라집니다.
5. 새 블록이 들어올 자리가 이미 막혀 있으면(천장까지 쌓임) 게임오버 배너가 뜹니다.
6. 다시 깃발 ▶ 을 누르면 새 게임이 시작됩니다.

## 룰

- **점수**: 한 번에 지운 줄 수에 따라 1줄 +100, 2줄 +300, 3줄 +500, 4줄 +800.
- **레벨**: 누적 줄수 ÷ 5 + 1. 레벨이 오를 때마다 낙하 간격이 0.1초씩 짧아집니다(0.8초 → 최소 0.12초).
- **블록 색**: I=시안, O=노랑, T=보라, S=초록, Z=빨강, J=파랑, L=주황 (7종 무작위).
- 최고기록은 점수가 갱신될 때 함께 기록됩니다.

## 작동 원리 (내부 구조)

추상적인 학습 콘셉트는 없지만, 2D 격자를 Scratch에서 다루는 방식이 핵심 설계 포인트입니다.

- **보드는 1차원 리스트**: 8열×14행 격자를 길이 112의 리스트로 평탄화합니다. 임의의 칸 (열 col, 행 row)은 인덱스 `idx = row*8 + col + 1` 로 읽고 씁니다. 충돌 검사·라인 검출·라인 제거가 전부 단순 인덱스 조회/치환이 됩니다.
- **회전은 룩업 테이블**: 행렬 연산 대신 7피스 × 4회전 × 4셀 = 112개의 셀 오프셋을 미리 계산해 두 리스트(오프셋C/오프셋R)에 담았습니다. (피스 p, 회전 r)의 셀 위치는 시작 인덱스 `((p-1)*4 + r)*4` 부터 4개를 읽어 기준칸에 더해 구합니다.
- **충돌 검사**: 가상으로 옮긴 블록의 4셀이 벽(col<0, col>7)·바닥(row>13) 밖이거나 이미 찬 칸과 겹치면 이동을 취소합니다.
- **라인 제거 끌어내림**: 꽉 찬 행을 따로 지우는 대신 그 행 위의 모든 칸을 한 칸 아래로 덮어쓰고 맨 윗줄을 비웁니다. 같은 행을 다시 검사해 여러 줄이 동시에 안전하게 처리됩니다.

## 실행

1. `테트리스미니.sb3` 다운로드
2. [scratch.mit.edu](https://scratch.mit.edu) → 만들기 → 파일 → 컴퓨터에서 가져오기
3. 초록 깃발 ▶ 클릭

## 다시 빌드

```bash
cd games/tetris-mini
python3 build.py
# → 테트리스미니.sb3 갱신
```

`테트리스미니.sb3` 파일이 생성됩니다. Scratch "내 작업실"에 업로드하거나 Scratch 데스크탑에서 열어주세요.

## 구조

- 스프라이트 5: Stage / 블록판정기 (게임 두뇌, 숨김) / 보드셀 (고정 블록 렌더 클론) / 현재블록 (떨어지는 4셀 클론) / 게임오버 배너
- 총 블록 484개 (Stage 48 / 블록판정기 342 / 보드셀 35 / 현재블록 47 / 게임오버 12)
- 커스텀 블록 2개: `충돌?`(경계+보드 충돌 검사), `고정`(셀 기록 + 라인 검출/끌어내림 + 점수)
- 리스트 5개: 보드(112) / 오프셋C(112) / 오프셋R(112) / 현셀X(4) / 현셀Y(4)
- 자산: 인라인 SVG 9종 (배경·셀 코스튬 7색·게임오버 배너) + pop.wav
