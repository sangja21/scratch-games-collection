# 도그파이트 (Dogfight)

> 두 비행기가 푸른 하늘에서 1대1로 맞붙는 공중전. 좌/우 회전과 항상 일정한 전진 속도, 정면 기관총만 있는 단순한 메커닉. 적기 AI 가 플레이어를 쫓아 회전하면서 사격하기 때문에 **상대 후방을 잡고 자기 후방은 안 보이게** 하는 것이 핵심입니다.

## 조작

- 왼쪽/오른쪽 화살표: 비행기 좌/우 회전 (틱당 4°)
- 비행기는 항상 자동으로 전진 (관성 없음, 일정 속도)
- 스페이스: 기관총 발사 (정면 직선, 쿨다운 약 0.25초)
- 화면 가장자리는 반대편으로 이어집니다 (비행기·총알 모두 wrap)

## 게임 흐름

1. 초록 깃발 클릭 → 파란 비행기(플레이어)는 왼쪽, 빨간 비행기(적)는 오른쪽에서 서로 마주보고 시작.
2. 적기는 매 틱 플레이어를 향해 회전하면서 약간의 흔들림을 가지고 따라옵니다. 1~2초마다 정면으로 사격.
3. 플레이어 체력 = 3, 적 체력 = 3. 총알에 닿을 때마다 −1.
4. **승리**: 적 체력 0 → YOU WIN! 배너.
5. **패배**: 플레이어 체력 0 → YOU LOSE 배너.

## 팁

- 적 후방으로 돌아가서 사격하면 적이 회피하기 어렵습니다.
- 적 정면을 그대로 들이받으면 서로 동시에 맞을 수 있어요. 옆구리·후방을 잡으세요.
- 화면 가장자리 wrap 을 이용해 빠르게 적의 등 뒤로 이동할 수 있습니다.
