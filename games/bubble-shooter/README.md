# 버블 슈터 (Bubble Shooter)

마우스로 조준하고 클릭해서 색깔 거품을 위로 쏜다. 같은 색 거품이 3개 이상 모이면 한꺼번에 터지면서 점수가 오른다. 거품은 좌우 벽에 맞으면 튕기고, 천장이나 기존 거품에 닿으면 가장 가까운 빈 칸에 딱 붙는다. 거품이 바닥선까지 내려오면 게임오버, 화면의 거품을 전부 없애면 승리. 1994년 Taito "퍼즐 보블"의 간소화 버전.

## 조작

| 입력 | 동작 |
|------|------|
| 마우스 이동 | 발사대 조준 (위쪽 반원 안에서만, 좌우 최대 80도) |
| 마우스 클릭 / 스페이스 | 현재 거품 발사 |
| 초록 깃발 ▶ | 새 게임 시작 / 재시작 |

발사대는 화면 아래(-105, -150)에 고정되어 마우스를 따라 회전합니다. 아래쪽으로는 조준되지 않고 항상 천장을 향해 위쪽으로만 쏩니다. 거품은 좌우 벽에서 튕기므로 벽 반사를 이용해 구석에 있는 거품도 맞힐 수 있습니다.

## 게임 흐름

1. 깃발을 누르면 8열 × 12행 격자의 위 5줄(40칸)이 빨강·주황·노랑·초록·파랑 5색 거품으로 무작위로 채워집니다.
2. 마우스로 겨눈 뒤 클릭하면 거품이 직선으로 날아갑니다 — 좌우 벽에서 반사, 천장이나 기존 거품에 닿으면 가장 가까운 빈 칸에 붙습니다.
3. 붙은 자리에서 같은 색 거품이 상하좌우로 3개 이상 연결되면 그 무리 전체가 터지고 점수가 오릅니다 (개당 +10점).
4. **5발마다 천장에서 거품이 한 줄씩 새로 내려옵니다** — 빨리 터뜨리지 않으면 점점 바닥에 가까워지는 압박 메커닉입니다.
5. 거품이 맨 아래 바닥선(빨간 선)에 닿으면 **GAME OVER**, 격자의 거품을 전부 없애면 **YOU WIN!** 배너가 뜹니다.
6. 깃발을 다시 누르면 즉시 새 게임이 시작됩니다.

## 룰

- 같은 색 3개 이상 연결 시 무리 전체가 터집니다 (점수 = 터진 개수 × 10).
- 우측 NEXT 박스에 다음에 쏠 거품 색이 미리 표시됩니다.
- 5발마다 한 줄이 추가로 내려와 압박이 커집니다.
- 거품이 바닥선(맨 아래 행)에 닿으면 게임오버.
- 격자에 남은 거품이 0이 되면 승리.
- 최고기록은 점수를 넘길 때마다 갱신됩니다.

## 실행

1. `버블슈터.sb3` 다운로드
2. [scratch.mit.edu](https://scratch.mit.edu) → 만들기 → 파일 → 컴퓨터에서 가져오기
3. 초록 깃발 ▶ 클릭

```bash
cd games/bubble-shooter

```

## 구조

- 스프라이트 5: Stage / 슈터(조준·비행·스냅·매칭의 두뇌) / 격자셀(붙은 거품 클론 ≤96) / NEXT 미리보기 / 배너
- 총 블록 509개 (Stage 27 / 슈터 417 / 격자셀 35 / NEXT 7 / 배너 23)
- 핵심 자료구조: `보드` 리스트 96칸(8열×12행 행 우선 평탄화, `idx = row*8 + col + 1`)이 격자 상태의 단일 진실원
- 핵심 알고리즘:
  - **발사**: 각도를 ±80도로 제한, `속도X = sin(각도)×12` / `속도Y = cos(각도)×12`로 분해, 좌우 벽에서 부호 반전 반사
  - **스냅**: 비행 거품 위치에서 빈 칸 중 제곱거리가 최소인 칸을 찾아 붙임
  - **매칭**: 스택 리스트 기반 반복 flood fill (재귀 없이 같은 색 상하좌우 연결 무리 탐색, 방문표시로 중복 방지, col7→col0 줄바꿈 가드)
- 자산: 인라인 SVG 8종 (배경·거품 5색·배너 2종) + pop.wav
