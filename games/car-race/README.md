# 자동차 레이싱 (Car Race)

좌/우 화살표로 내 차를 한 차선씩 옮겨, 위에서 내려오는 적 차를 피한다. 도로 페인트 라인이 위→아래로 흘러 속도감을 표현한다. 시간이 지날수록 적 차 스폰 주기가 짧아지고 모든 게 더 빨라진다.

## 조작

| 키 | 동작 |
|----|------|
| ← | 한 차선 왼쪽으로 이동 |
| → | 한 차선 오른쪽으로 이동 |

4차선 도로(차선 중심 x = -90 / -30 / +30 / +90)를 따라 스냅 이동합니다. 길게 눌러도 한 번에 한 차선만 이동하고, 손을 떼었다 다시 눌러야 다음 차선으로 갑니다.

## 룰

- 내 차가 적 차에 닿으면 게임오버.
- 점수는 생존 시간 (매 초 +1).
- 5초마다: 스폰 주기 0.88배, 적 차 낙하 속도 +0.6 (상한 9), 페인트 라인 스크롤 속도 +0.8 (상한 12).
- 적 차는 빨강·파랑·노랑 3색이 무작위로 등장.
- 최고기록은 게임오버 시 갱신됩니다.

## 실행

1. `자동차_레이싱.sb3` 다운로드

3. 초록 깃발 ▶ 클릭

