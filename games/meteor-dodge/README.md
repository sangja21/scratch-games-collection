# 운석 회피 (Meteor Dodge)

우주선을 화살표로 자유 이동시켜 화면 위에서 떨어지는 운석을 피한다. 오래 살아남을수록 점수가 오른다. 시간이 지날수록 운석 생성 주기가 짧아져 난이도가 점점 올라간다.

## 조작

| 키 | 동작 |
|----|------|
| ← | 왼쪽 이동 |
| → | 오른쪽 이동 |
| ↑ | 위쪽 이동 |
| ↓ | 아래쪽 이동 |

화면 가장자리(좌/우/상/하)에서는 멈춥니다 (clamp). 관성 없음, 직접 위치 변경.

## 룰

- 우주선이 운석에 닿으면 게임오버.
- 점수는 생존 시간 (매 초 +1).
- 5초마다 운석 스폰 주기가 0.85배 짧아져 점점 더 정신없어집니다 (하한 0.25초).
- 운석은 큰(2.5)·중(3.5)·작(4.5) 세 종류가 무작위로 떨어집니다. 작은 운석이 더 빠릅니다.
- 최고기록은 게임오버 시 갱신됩니다.

## 실행

1. `운석_회피.sb3` 다운로드
2. [scratch.mit.edu](https://scratch.mit.edu) → 만들기 → 파일 → 컴퓨터에서 가져오기
3. 초록 깃발 ▶ 클릭

## 다시 빌드

```bash
cd games/meteor-dodge
python3 build.py
# → 운석_회피.sb3 갱신
```
