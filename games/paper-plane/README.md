# 종이비행기 멀리 던지기 (Paper Plane)

화면 왼쪽 언덕에서 종이비행기를 **각도 → 힘** 2단 타이밍 게이지로 던진다. 던진 비행기는 중력으로 천천히 미끄러져 내려오며(글라이딩), 매 턴 바뀌는 **바람**(순풍이면 멀리, 역풍이면 짧게)에 밀린다. 비행기는 화면 안에서 날고, 멀리 간 만큼 배경과 거리 깃발이 왼쪽으로 흘러(카메라) 거리를 표현한다. 총 3번 던져 합계와 베스트를 겨룬다.

## 조작

| 키 | 동작 |
|----|------|
| 스페이스 (1번째) | 왕복하는 **각도 바늘**(0°~80°)을 원하는 각도에서 고정 |
| 스페이스 (2번째) | 차오르락 내리락 하는 **힘 게이지**(0~100)를 원하는 세기에서 멈춰 발사! |

스페이스 단 하나로 플레이합니다. 한 번 누르면 각도가 고정되고, 다시 누르면 그 힘으로 날아갑니다. (같은 한 번 누름이 두 단계를 한꺼번에 통과하지 않도록, 키를 뗐다 다시 눌러야 다음 단계로 갑니다.)

## 게임 흐름

1. 초록 깃발 ▶ → 거리·베스트·합계 0, 남은 던지기 3으로 시작.
2. **던지기 준비**: 이번 턴 바람이 새로 정해지고(상단 중앙 화살표 → 순풍 / ← 역풍, 크기 = 세기), 비행기가 발사대로 돌아옵니다.
3. **각도 맞추기**: 바늘이 0°~80°를 왕복 → 스페이스로 고정.
4. **힘 맞추기**: 좌하단 힘 게이지가 왕복 → 스페이스로 발사.
5. **비행**: 비행기가 글라이딩하며 바람에 밀려 날아갑니다. 멀리 갈수록 배경/깃발이 왼쪽으로 흘러 비행기가 화면 밖으로 사라지지 않습니다.
6. **착지**: 지면에 닿으면 이번 거리가 확정 → 합계에 더해지고, 베스트를 넘으면 "신기록!" 배너.
7. 3번 모두 던지면 **결과 배너**. 초록 깃발로 다시 도전.

## 조작 팁 (학습 포인트는 없음 — 순수 액션)

- **각도와 힘**은 "위로 얼마나, 세게 얼마나"라는 직관입니다(내부적으로 각도·힘이 가로/세로 속도로 바뀌지만 화면에는 미터와 바람만 보입니다).
- 바람 화살표를 보고 던지기를 조절하세요. **순풍(→)이면 비행기가 훨씬 멀리 가고, 역풍(←)이면 짧게 떨어집니다.** 같은 던지기라도 바람이 결과를 크게 좌우하니, 순풍 턴을 잘 살리는 게 합계를 키우는 열쇠입니다.
- 바람이 없을 때 가장 멀리 가는 각도는 대략 **45° 부근**입니다.
