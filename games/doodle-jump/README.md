# 🦘 두들 점프 (Doodle Jump)

발판을 밟고 끝없이 위로 점프! 2009년 모바일 클래식 Doodle Jump 의 Scratch 이식.

## 🎮 조작

| 키 | 동작 |
|----|------|
| ← | 왼쪽 이동 |
| → | 오른쪽 이동 |
| (자동) | 발판에 닿으면 자동 점프 |

화면 가장자리를 벗어나면 반대편에서 다시 등장합니다 (좌↔우 wrap).

## 🎯 룰

- 발판을 밟으면 위로 튕깁니다.
- 캐릭터가 화면 중앙 위로 올라가면 모든 발판이 아래로 스크롤됩니다 (= 카메라가 위로 따라옴).
- 발판을 놓치고 화면 바닥으로 떨어지면 게임오버.
- 점수는 누적 스크롤 거리 (= 올라간 높이). 최고기록은 게임오버 시 갱신됩니다.

## ▶ 실행

1. `두들_점프.sb3` 다운로드
2. [scratch.mit.edu](https://scratch.mit.edu) → 만들기 → 파일 → 컴퓨터에서 가져오기
3. 초록 깃발 ▶ 클릭

## 🛠 다시 빌드

```bash
cd games/doodle-jump
python3 build.py
# → 두들_점프.sb3 갱신
```
