# 미사일 커맨드 (Missile Command)

1980년 아타리 클래식 *Missile Command* 클론. 도시 3곳을 지키며 떨어지는 적 미사일을 **마우스 클릭**으로 요격하세요.

## 🎮 플레이

1. 초록 깃발(▶) 클릭으로 시작.
2. 빨간 십자선이 마우스를 따라다닙니다.
3. 화면 어디든 **클릭**하면 그 자리에서 폭발이 일어나 주변 적 미사일을 격추.
4. (또는 스페이스바를 눌러도 마우스 위치에서 폭발 발사.)
5. 도시 3곳(`아래쪽 좌·중·우`)이 적 미사일에 맞으면 라이프 1 감소 + 도시 폐허화.
6. 라이프 0 이 되면 **GAME OVER**.
7. 라운드를 진행할수록 미사일이 더 많이, 더 빠르게 떨어집니다.

## 🧱 메커닉

| 요소 | 값 |
|------|----|
| 초기 라이프 (도시 수) | 3 |
| 라운드 1 적 미사일 수 | 6 |
| 라운드당 미사일 +2 | 라운드↑ 마다 +2 |
| 적 속도 | 라운드당 +0.3 |
| 격추 점수 | 미사일당 +25 |
| 폭발 지속 | 약 0.55초 (확장 0.3s + 수축 0.25s) |

## 🛠 다시 빌드

```bash
cd games/missile-command
python3 build.py     # → 미사일_커맨드.sb3 갱신
```

Python 3 만 있으면 외부 라이브러리 없이 동작합니다.
