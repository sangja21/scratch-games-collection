# 폭탄 스쿼드

> 하단의 슬링샷을 마우스로 **누르고 당겨(드래그)** 포탄을 **포물선**으로 쏴 올려, 하늘에서 낙하산 타고 천천히 내려오는 **적 몬스터를 요격**하는 드래그 조준 물리 방어 게임(닌텐도 *Bob-omb Squad* 오마주). 당기는 동안 **점선 궤적**이 실시간으로 휘어, 떨어지는 적의 '앞'을 겨눠 맞힌다. 마우스를 놓으면 점선 그대로 포탄이 날아가 적을 터뜨린다 — 한 발이 여러 적을 **꿰뚫는 관통샷**에, 이전 발이 날아가는 중에도 **계속 쏘는 연사**까지. 바닥 **꽃밭(꽃 = 라이프)** 에 적이 닿으면 꽃이 하나 시들고, 꽃이 다 사라지면 게임오버. 연속으로 맞히면 **콤보 점수가 배로** 불어나고, 시간이 갈수록 적이 더 빨리·더 자주 떨어진다 — **최고기록에 도전하는 엔들리스**.

배경음악(BGM)이 깔린 채로, 이 게임의 진짜 재미는 **"코드의 숫자를 직접 바꿔보며 내 게임으로 만드는 것"** 이다. 발사력·중력·바람 같은 물리 상수부터 낙하속도·스폰간격·꽃 개수·콤보 배율·관통·연사·BGM 음량까지 **24개의 조절 값이 모두 한글 변수**로 정리돼 있어, 숫자 하나만 바꿔도 게임이 확 달라진다. 특히 **`중력` 하나**로 "거의 직선으로 날아가는 쉬운 슬링샷"과 "활처럼 크게 휘는 어려운 로빙"을 오갈 수 있다. 아래 [개조 손잡이](#️-개조-손잡이-바꿔보세요)와 [개조 미션 사다리](#-개조-미션-사다리)를 따라 하나씩 도전해 보자.

## 조작법

| 입력 | 동작 |
|------|------|
| **발사체를 마우스로 누른 채 당기기(드래그)** | 하단 슬링샷의 발사체를 잡아 **당긴 반대 방향**으로 발사 벡터가 정해진다. 멀리 당길수록 세게(단, `최대당김거리`까지만) |
| **마우스 놓기** | 그 순간의 점선 궤적대로 포탄이 **발사**된다. 탄약은 무제한 — 이전 발이 아직 공중에 있어도 곧바로 **연사**로 다음 발을 쏠 수 있다(동시에 `최대포탄수`발까지) |
| (자동 · 관통) | 포탄 한 발은 적 하나를 맞혀도 사라지지 않고 **`관통횟수`만큼 적을 꿰뚫는다** — 일렬로 떨어지는 적들을 한 발로 여럿 잡을 수 있다 |
| (자동) | 당기는 동안 **흰 점선 궤적**이 "지금 놓으면 어디로 날아갈지"를 미리 그려준다. 적 낙하선과 점선이 만나는 지점을 겨누면 명중 |
| 초록 깃발 | 시작 / 다시 시작 (점수·콤보·꽃·난이도 리셋. **최고점수는 유지**된다) |

> 좌상단 HUD 에 `점수`·`콤보배수`·`최고점수` 3종과, 드래그 벡터 수치 `당김X`·`당김Y`·`당김크기`·`당김각도` 4종이 실시간으로 보인다. **당김X·당김Y 가 곧 포탄의 옆·위 속도 성분**이다 — 당기면서 숫자가 어떻게 변하는지 관찰해 보자.

## 게임 규칙

1. **시작** — 초록 깃발을 누르면 모든 값이 기본값으로 맞춰지고 **배경음악(BGM)이 흐르기 시작한다**(곡이 끝나면 처음부터 무한 반복). 하늘·지면·꽃밭이 그려지고, 바닥엔 꽃 4개(라이프)가 나란히 핀 뒤, 곧 하늘 위에서 낙하산 단 적 몬스터가 내려오기 시작한다.
2. **조준 & 발사(연사)** — 하단 슬링샷의 발사체를 눌러 당기면(고무줄 '삐긋' 소리) 당긴 반대 방향으로 점선 궤적이 실시간으로 휜다. 마우스를 놓으면 '핑!' 하고 포탄이 포물선으로 날아간다. 이전 발이 아직 공중에 있어도 계속 쏠 수 있어(동시에 최대 `최대포탄수`발), 우르르 몰려오는 적을 촘촘히 요격한다.
3. **요격 & 관통 & 콤보** — 포탄이 적에 닿으면 '펑!' 터지며 점수가 오른다. 포탄은 사라지지 않고 **`관통횟수`만큼 적을 꿰뚫어**, 일렬로 떨어지는 적을 한 발로 여럿 잡을 수 있다. **연속으로 맞힐수록 콤보배수가 배로**(1→2→4→8…, `콤보최대`까지) 뛰어 점수가 폭발한다. 명중 자리에 금색 숫자가 떠오른다.
4. **놓침 & 라이프** — 적이 요격되지 않고 꽃밭 라인에 닿으면 가장 가까운 꽃 하나가 시들고(콤보도 0으로 리셋), 남은 꽃이 줄어든다.
5. **라키투 보너스** — 가끔 하늘을 가로지르는 **라키투**가 등장한다(`1/라키투확률` 확률). 이걸 맞히면 '콰광!' 화면의 **모든 적이 한 번에 전멸**하고 큰 보너스 점수를 준다.
6. **난이도 램프** — 적을 잡을수록(처치수↑) 낙하속도가 오르고 스폰간격이 줄어(`난이도증가율`) 점점 빡세진다. 스테이지·클리어는 없는 **무한 생존**이다.
7. **게임오버 → 최고기록** — 꽃이 다 사라지면 게임오버 배너가 뜬다. 이번 점수가 최고점수를 넘으면 '신기록' 팡파르가 울리고 기록이 갱신된다(재시작해도 최고점수는 유지). 초록 깃발을 다시 누르면 처음부터.

## ⚙️ 개조 손잡이 (바꿔보세요!)

게임을 바꾸는 모든 숫자는 **무대(Stage)의 초록 깃발 스크립트 한 곳**에 모여 있다. 코드 어디에도 숨겨진 숫자가 없으니, 여기 24개 손잡이만 돌리면 게임이 달라진다.

> 찾는 법: Scratch 에서 **무대**를 클릭 → 코드 탭 → 맨 위 `초록 깃발을 클릭했을 때` 묶음을 보면 `중력 을(를) 0.3 로 정하기` 같은 블록이 쭉 있다. 그 숫자만 바꾸면 끝! (노란 코멘트 "🛠️ 개조 손잡이" 가 붙어 있다.)

### 조준 / 물리 손잡이 (★ 이 4개가 궤적과 손맛을 바꾼다 — 특히 중력)

| 손잡이 (변수) | 기본값 | 무슨 뜻? / 바꾸면 일어나는 일 |
|---------------|:------:|-------------------------------|
| **발사력배율** | 0.22 | 당긴 거리를 초기 속도로 바꾸는 배율(초기 속도 = 당김 벡터 × 이 값). 올리면 살짝만 당겨도 포탄이 쭉 날아간다 |
| **최대당김거리** | 90 | 당김을 제한하는 최대 거리(px). 이보다 멀리 당겨도 힘은 이 값에서 멈춘다(최대 발사력 상한) |
| **중력** | 0.30 | 아래로 당기는 가속. **★손맛+난이도 동시 손잡이!** 낮으면(0.1) 포탄이 거의 **직선으로 멀리**(쉬움, Bob-omb 느낌), 높으면(0.7) **활처럼 크게 휘어**(어려움, 로빙) 리드 조준이 필요하다 |

## 🎯 개조 미션 사다리

쉬운 것부터 어려운 것까지. **하나씩 깨보면서 "코드를 망가뜨리고 고치는" 재미**를 느껴보자. 망가져도 괜찮다 — 초록 깃발만 다시 누르면 된다!

### 0층 — 숫자 하나 바꾸기 (무대 초록 깃발 스크립트)
- **달나라 로빙 요격**: `중력` 을 `0.30` → `0.10` 으로. 포탄이 거의 직선으로 멀리 날아가 조준이 쉬워진다(Bob-omb 느낌).
- **목성 활쏘기**: `중력` 을 `0.30` → `0.70` 으로. 포탄이 활처럼 크게 휘어 리드 조준이 필요(하드).
- **하드모드 꽃 1개**: `꽃개수` 를 `4` → `1` 로. 한 번만 뚫려도 게임오버 — 목숨을 건 요격.
- **여유 꽃밭**: `꽃개수` 를 `4` → `8` 로. 실수해도 오래 버틴다.
- **적 러시**: `스폰간격` 을 `1.6` → `0.6` 으로. 적이 우르르 쏟아진다(난이도 급상승).
- **슬로 적**: `낙하속도` 를 `0.7` → `0.3` 으로. 적이 아주 천천히 내려와 초보도 쉽게.
- **점수 폭발**: `콤보배율` 을 `2` → `3` 으로(+`콤보최대` 를 `81` 로). 콤보가 붙으면 점수가 어마어마하게.
- **지그재그 적**: `흔들폭` 을 `12` → `40` 으로. 적이 심하게 좌우로 흔들려 맞히기 어렵다.
- **라키투 자주**: `라키투확률` 을 `40` → `10` 으로. 라키투가 자주 나와 전멸 보너스 찬스가 늘어난다.
- **힘센 슬링샷**: `발사력배율` 을 `0.22` → `0.35` 로. 살짝만 당겨도 포탄이 쭉.
- **관통 마스터**: `관통횟수` 를 `2` → `5` 로. 한 발이 다섯 마리를 꿰뚫어 일렬로 떨어지는 적을 쓸어담는다.
- **탄막 연사**: `최대포탄수` 를 `5` → `12` 로. 화면에 포탄을 잔뜩 띄워 쉴 새 없이 쏜다(반대로 `1` 로 낮추면 한 발씩 신중하게).
- **긴 사거리 포탄**: `포탄수명` 을 `150` → `400` 으로. 포탄이 더 오래 날아 멀리·높이 있는 적까지 노린다(너무 크게 하면 화면 밖 포탄이 오래 남는다).
- **조용한 배경음**: `브금볼륨` 을 `70` → `20` 으로. 배경음악을 낮춰 요격·콤보 효과음을 또렷하게 즐긴다.

### 1층 — 값을 식으로 묶기 (관계 만들기)
- **콤보 비례 연출**: 팝업 크기를 `콤보배수` 에 비례시켜(예: 크기 = 30 + 콤보배수 × 2) 콤보가 클수록 숫자가 커 보이게.
- **가팔라지는 난이도**: `난이도증가율` 을 `0.02` → `0.05` 로. 후반이 훨씬 빡세진다(실제 낙하 = 낙하속도 + 처치수 × 이 값).
- **처치 비례 보너스**: 요격 점수에 `+ 처치수` 를 더해 오래 살수록 한 방 점수가 커지게 — 스테이지 없이도 성장감.

### 2층 — if 조건 더하기
- **황금 적**: 적 스폰에 `만약 (1~12 난수 = 1) → 코스튬 금색·이번점수 × 3` 분기(가끔 3배 점수 적).
- **위기 경고**: 무대에 `만약 꽃남음 = 1 → 배경 붉게/사이렌음` 으로 마지막 꽃 경고.
- **콤보 마일스톤**: 적 요격에서 `만약 콤보수 mod 10 = 0 → 점수 +1000` 보너스.

### 3층 — 규칙 자체를 바꾸기 (분기)
- **바람 부는 날**: `바람` 을 `0` → `0.08` 로(또는 매판 `바람 = -8~8 난수 / 100` 랜덤) — 궤적이 휘어 예측이 달라진다.
- **튕기는 포탄**: 포탄 비행에 `만약 y>190 → 속도Y = 속도Y × -0.6`(1회) → 천장에 튕겨 위쪽 적 노리기.
- **꽃 부활**: `만약 처치수 mod 30 = 0 이고 꽃남음 < 꽃개수 → 시든 꽃 1개 되살리기` — 오래 버티면 라이프 회복.

### 4층 — 새 메커닉 만들기 (구조)
- **꽃밭 재배치**: `꽃X` 리스트 좌표·`꽃개수` 를 바꿔 꽃을 몰아 두거나 넓게 퍼뜨려 방어 난이도를 바꾸자.
- **2단 낙하산**: 적이 아래 절반에서 낙하산을 접고 **가속 낙하**(내낙하 × 2)하게 만들어 후반 압박을 키우자.
- **연쇄 폭발**: 적 요격 시 주변 적도 같이 터지는 광역(콤보 폭증) 메커닉 추가.
- **당김 가이드 원**: 발사대에 `최대당김거리` 반경 원(펜)을 그려 최대 힘 지점을 시각화.

## 학습 포인트

**포물선 운동·벡터·이동표적 리드를 손으로 당겨 쏘며 만나는 물리 방어 게임.** 발사체를 당기면 **당김 벡터가 곧 발사 벡터**(당긴 반대 방향)가 되고, 놓으면 그 점선 궤적대로 포탄이 날아가 떨어지는 적을 요격한다. 액션의 *결과로* 물리와 수학이 슬쩍 드러난다:

- **↗️ 속도 성분 분해 (벡터)** — 당김 벡터 `(당김X, 당김Y)` 가 그대로 초기 속도 성분 `속도X = 당김X × 발사력배율`(옆)·`속도Y = 당김Y × 발사력배율`(위)이 된다. 위로 당기면 위 성분↑·옆 성분↓ — 드래그하며 X·Y 성분을 직접 만지고 HUD `당김X`/`당김Y` 로 확인한다. 대각선 45°로 당기면 두 성분이 같아져 사거리가 최대(바람 0).
- **🌙 중력의 적분적 영향 = 손맛/난이도** — 매 틱 `속도Y ← 속도Y − 중력` 이라 포탄이 올라갔다 떨어진다. **`중력` 하나로 게임 성격이 바뀐다**: 낮으면 정점이 얕고 궤적이 곧아 **직선 슬링샷(쉬움)**, 높으면 크게 휘는 **로빙(어려움)**. 물리 상수가 곧 난이도라는 걸 몸으로 안다.
- **🎯 이동표적 리드 (예측 / 상대운동)** — 적은 계속 내려오는 표적이라, 지금 위치가 아니라 **포탄이 도달할 때쯤 적이 있을 자리(앞)** 를 겨눠야 맞는다. 발사부터 명중까지의 시간 동안 적이 얼마나 더 내려올지 예측하는 **리드 사격**을 자연히 익힌다 — 점선 궤적과 적 낙하선이 만나는 지점을 눈으로 맞추는 놀이.
- **✏️ 예측 = 시뮬레이션** — 궤적펜은 실제 포탄과 **똑같은 물리식을 미리 돌려(dry-run)** 점선으로 보여준다. "계산해서 미리 안다"는 물리·프로그래밍 공통 아이디어를 눈으로 만난다.

부수적으로 **변수·리스트·조건·클론·펜·벡터 연산·최솟값 선택(가장 가까운 꽃)·콤보 배증(지수적 증가)** 을 게임을 튜닝하며(0→4층) 코딩 학습으로 이어간다. 코드 곳곳의 **노란 코멘트(가이드 투어)** 가 각 부분을 설명한다. 어디까지나 직관적인 드래그 방어 놀이로 포장된 채 — 단일 슬링샷·명확한 표적(낙하하는 적)이라는 좋은 게임 모양을 지킨다.
